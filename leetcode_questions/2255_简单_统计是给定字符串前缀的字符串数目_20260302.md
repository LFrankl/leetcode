## 题目描述

<p>给你一个字符串数组&nbsp;<code>words</code>&nbsp;和一个字符串&nbsp;<code>s</code>&nbsp;，其中&nbsp;<code>words[i]</code> 和&nbsp;<code>s</code>&nbsp;只包含 <strong>小写英文字母</strong>&nbsp;。</p>

<p>请你返回 <code>words</code>&nbsp;中是字符串 <code>s</code>&nbsp;<strong>前缀&nbsp;</strong>的 <strong>字符串数目</strong>&nbsp;。</p>

<p>一个字符串的 <strong>前缀</strong>&nbsp;是出现在字符串开头的子字符串。<strong>子字符串</strong>&nbsp;是一个字符串中的连续一段字符序列。</p>

<p>&nbsp;</p>

<p><strong>示例 1：</strong></p>

<pre><b>输入：</b>words = ["a","b","c","ab","bc","abc"], s = "abc"
<b>输出：</b>3
<strong>解释：</strong>
words 中是 s = "abc" 前缀的字符串为：
"a" ，"ab" 和 "abc" 。
所以 words 中是字符串 s 前缀的字符串数目为 3 。</pre>

<p><strong>示例 2：</strong></p>

<pre><b>输入：</b>words = ["a","a"], s = "aa"
<b>输出：</b>2
<strong>解释：
</strong>两个字符串都是 s 的前缀。
注意，相同的字符串可能在 words 中出现多次，它们应该被计数多次。</pre>

<p>&nbsp;</p>

<p><strong>提示：</strong></p>

<ul>
	<li><code>1 &lt;= words.length &lt;= 1000</code></li>
	<li><code>1 &lt;= words[i].length, s.length &lt;= 10</code></li>
	<li><code>words[i]</code> 和&nbsp;<code>s</code>&nbsp;<strong>只</strong>&nbsp;包含小写英文字母。</li>
</ul>

## 题目分析
我们需要统计数组 `words` 中有多少个字符串是给定字符串 `s` 的前缀。  
**前缀**：字符串 `s` 从开头开始的连续子串。  
注意：`words` 中可能有重复字符串，每个都要单独计数。

---

## 解题思路

### 核心判断
对于 `words` 中的每个字符串 `w`，判断它是否是 `s` 的前缀，即：
1. `len(w) <= len(s)`
2. `s[0:len(w)] == w`

---

## 解法一：直接遍历比较
最直观的方法：遍历 `words`，对每个单词检查它是否是 `s` 的前缀。

**时间复杂度**：  
- 每个单词长度 ≤ 10，s 长度 ≤ 10  
- 每次比较最多比较 10 个字符  
- 总复杂度 O(n * L)，n ≤ 1000，L ≤ 10，所以非常快。

**空间复杂度**：O(1)，只用了常数空间。

---

### 解法一代码实现

```go
func countPrefixes(words []string, s string) int {
    count := 0
    for _, w := range words {
        if len(w) <= len(s) && s[:len(w)] == w {
            count++
        }
    }
    return count
}
```

```cpp
class Solution {
public:
    int countPrefixes(vector<string>& words, string s) {
        int count = 0;
        for (const string& w : words) {
            if (w.size() <= s.size() && s.substr(0, w.size()) == w) {
                count++;
            }
        }
        return count;
    }
};
```

---

## 解法二：使用字符串查找函数
利用语言内置的 `strings.HasPrefix`（Go）或 `s.find(w) == 0`（C++）来判断前缀。

**时间复杂度**：与解法一相同 O(n * L)。  
**空间复杂度**：O(1)。

---

### 解法二代码实现

```go
func countPrefixes(words []string, s string) int {
    count := 0
    for _, w := range words {
        if strings.HasPrefix(s, w) {
            count++
        }
    }
    return count
}
```

```cpp
class Solution {
public:
    int countPrefixes(vector<string>& words, string s) {
        int count = 0;
        for (const string& w : words) {
            if (s.find(w) == 0) {
                count++;
            }
        }
        return count;
    }
};
```

## 解法三：前缀哈希集合（如果 s 很长且 words 很多时优化）
对于本题数据规模很小，不需要优化，但可以展示另一种思路：  
预先计算 `s` 的所有前缀，存入哈希集合，然后判断每个单词是否在集合中。

**时间复杂度**：  
- 构建前缀集合 O(L²)，L ≤ 10  
- 查询每个单词 O(1)  
- 总复杂度 O(L² + n)

**空间复杂度**：O(L²) 存储所有前缀。

---

### 解法三代码实现

```go
func countPrefixes(words []string, s string) int {
    prefixSet := make(map[string]bool)
    for i := 1; i <= len(s); i++ {
        prefixSet[s[:i]] = true
    }
    count := 0
    for _, w := range words {
        if prefixSet[w] {
            count++
        }
    }
    return count
}
```

```cpp
class Solution {
public:
    int countPrefixes(vector<string>& words, string s) {
        unordered_set<string> prefixSet;
        for (int i = 1; i <= s.size(); i++) {
            prefixSet.insert(s.substr(0, i));
        }
        int count = 0;
        for (const string& w : words) {
            if (prefixSet.count(w)) {
                count++;
            }
        }
        return count;
    }
};
```
