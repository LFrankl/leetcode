<p>你必须设计并实现时间复杂度为&nbsp;<code>O(log n)</code>&nbsp;的算法解决此问题。</p>

<p>&nbsp;</p>

<pre>

<pre>
<strong>输入：</strong>nums = [<code>5,7,7,8,8,10]</code>, target = 6

<p><strong>示例 3：</strong></p>

<strong>输入：</strong>nums = [], target = 0
<strong>输出：</strong>[-1,-1]</pre>

<p>&nbsp;</p>

## AI 解答 (DeepSeek)

### 解法一：两次二分查找

思路：由于数组有序且要求 O(log n)，核心是用二分查找分别定位 target 的左边界和右边界。找左边界时，当 nums[mid] >= target 时收缩右边界；找右边界时，当 nums[mid] <= target 时收缩左边界。最后验证找到的位置是否确实等于 target。

```go
func searchRange(nums []int, target int) []int {
    left := findLeft(nums, target)
    if left == -1 {
        return []int{-1, -1}
    }
    right := findRight(nums, target)
    return []int{left, right}
}

func findLeft(nums []int, target int) int {
    lo, hi := 0, len(nums)-1
    idx := -1
    for lo <= hi {
        mid := lo + (hi-lo)/2
        if nums[mid] >= target {
            hi = mid - 1
        } else {
            lo = mid + 1
        }
        if nums[mid] == target {
            idx = mid
        }
    }
    return idx
}

func findRight(nums []int, target int) int {
    lo, hi := 0, len(nums)-1
    idx := -1
    for lo <= hi {
        mid := lo + (hi-lo)/2
        if nums[mid] <= target {
            lo = mid + 1
        } else {
            hi = mid - 1
        }
        if nums[mid] == target {
            idx = mid
        }
    }
    return idx
}
```

复杂度：时间 O(log n)，空间 O(1)

### 解法二：二分查找目标值与目标值+1的左边界

思路：更简洁的写法——写一个辅助函数 `lowerBound(nums, target)` 返回第一个 >= target 的下标。左边界即 `lowerBound(nums, target)`，右边界即 `lowerBound(nums, target+1) - 1`。通过验证左边界是否合法来判断 target 是否存在。

```go
func searchRange(nums []int, target int) []int {
    l := lowerBound(nums, target)
    if l == len(nums) || nums[l] != target {
        return []int{-1, -1}
    }
    r := lowerBound(nums, target+1) - 1
    return []int{l, r}
}

func lowerBound(nums []int, target int) int {
    lo, hi := 0, len(nums)
    for lo < hi {
        mid := lo + (hi-lo)/2
        if nums[mid] < target {
            lo = mid + 1
        } else {
            hi = mid
        }
    }
    return lo
}
```

复杂度：时间 O(log n)，空间 O(1)
