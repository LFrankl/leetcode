## AI 解答 (DeepSeek)

## 1️⃣ 题意理解

**题目重述**  
给定一个表示正整数的字符串 `num`，要求移除字符串末尾的所有零（尾随零），并以字符串形式返回结果。

**关键约束**  
- 字符串长度范围：1 ≤ num.length ≤ 1000  
- 字符串仅包含数字 '0'-'9'  
- 字符串不含前导零（即不会出现 "0123" 这样的输入）  
- 需要保留非零部分，如果整个字符串都是零（如 "0"），应返回 "0"（但题目保证是正整数，所以不会出现全零）

**边界情况**  
1. 字符串末尾没有零 → 直接返回原字符串  
2. 字符串末尾有多个零 → 移除所有尾随零  
3. 字符串全为零 → 但题目说 num 是正整数，所以不会出现  
4. 字符串长度为 1 → 直接返回  

**输入输出格式**  
- 输入：字符串 `num`  
- 输出：移除尾随零后的字符串  

---

## 2️⃣ 解题思路

### 解法一：反向遍历
**核心思想**  
从字符串末尾开始向前遍历，找到第一个非零字符的位置，然后截取从开头到该位置的子串。

**步骤**  
1. 初始化指针 `i` 指向字符串末尾  
2. 从后向前移动指针，跳过所有零字符  
3. 当遇到非零字符或到达字符串开头时停止  
4. 截取 `num[0:i+1]` 作为结果  
5. 如果全部是零（理论上不会发生），返回 "0"

**注意事项**  
- 注意索引边界，避免越界  
- 如果字符串全零，需要特殊处理（但题目保证不会出现）

---

### 解法二：内置函数/字符串处理
**核心思想**  
利用字符串处理函数（如 `strings.TrimRight`）直接移除尾随零。

**步骤**  
1. 使用 `strings.TrimRight(num, "0")` 移除所有尾随零  
2. 如果结果为空字符串（全零情况），返回 "0"  
3. 否则返回处理后的字符串

**注意事项**  
- 需要处理全零的特殊情况  
- 虽然代码简单，但依赖语言内置函数

## 3️⃣ 代码实现

### 解法一：反向遍历
```go
func removeTrailingZeros(num string) string {
    // 从末尾开始找到第一个非零字符的位置
    i := len(num) - 1
    for i >= 0 && num[i] == '0' {
        i--
    }
    // 如果 i < 0 说明整个字符串都是零（理论上不会发生）
    if i < 0 {
        return "0"
    }
    // 截取从开头到第一个非零尾随字符的子串
    return num[:i+1]
}
```

### 解法二：内置函数
```go
import "strings"

func removeTrailingZeros(num string) string {
    // 使用 TrimRight 移除所有尾随零
    result := strings.TrimRight(num, "0")
    // 如果结果为空，说明原字符串全是零
    if result == "" {
        return "0"
    }
    return result
}
```

## 4️⃣ 复杂度分析

### 解法一：反向遍历
- ⏱️ **时间复杂度**：O(n)  
  最坏情况下需要遍历整个字符串（当没有尾随零时），平均情况只需遍历尾随零部分。

- 💾 **空间复杂度**：O(1)  
  只使用了常数级别的额外空间（指针 i），返回的子串是原字符串的切片（Go 中字符串切片不分配新内存）。

### 解法二：内置函数
- ⏱️ **时间复杂度**：O(n)  
  `strings.TrimRight` 内部实现也是遍历字符串，时间复杂度为 O(n)。

- 💾 **空间复杂度**：O(n)  
  `strings.TrimRight` 返回的是新字符串，需要分配新内存，空间复杂度为 O(n)。

3. 对于本题的简单需求，手动实现更直观
