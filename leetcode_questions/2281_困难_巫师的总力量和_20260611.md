## 巫师的总力量和（LeetCode 2281）

### 简要思路

对于每个元素 `strength[i]`，计算它作为子数组最小值时的总贡献 = `strength[i] × Σ(sum(subarray))`。用单调栈找到每个元素作为最小值的左右边界，再用**前缀和的前缀和**将嵌套求和化为 O(1) 计算，从而整体 O(n) 解决。

---

### 解法一：贡献法 + 单调栈 + 二重前缀和

**思路：**

核心思想是**贡献法**：总答案 = Σ 每个元素 `v=strength[i]` 作为最小值时的贡献。

1. **找边界**：用单调栈找每个元素左边**严格更小**的位置 `L` 和右边**更小或相等**的位置 `R`，确保每个子数组只被一个最小值负责（避免重复计数）。
2. **计算贡献**：`strength[i]` 是 `[l, r]` 的最小值，其中 `L < l ≤ i ≤ r < R`。贡献为：

$$v \times \sum_{l=L+1}^{i}\sum_{r=i}^{R-1}(\text{prefix}[r+1] - \text{prefix}[l])$$

展开后：

$$v \times \big[(i-L) \cdot \text{pprefix}[R+1] - \text{pprefix}[i+1] - (R-i) \cdot \text{pprefix}[i+1] - \text{pprefix}[L+1]\big]$$

其中 `pprefix` 是前缀和的前缀和，使所有求和 O(1) 完成。

```go
func totalStrength(strength []int) int {
    n := len(strength)
    mod := int64(1e9 + 7)

    // 前缀和 prefix[k] = Σ strength[0..k-1]
    prefix := make([]int64, n+1)
    for i := 0; i < n; i++ {
        prefix[i+1] = (prefix[i] + int64(strength[i])) % mod
    }

    // 前缀和的前缀和 pprefix[k] = Σ prefix[0..k-1]
    pprefix := make([]int64, n+2)
    for i := 0; i <= n; i++ {
        pprefix[i+1] = (pprefix[i] + prefix[i]) % mod
    }

    // 单调栈：左边界 L（前一个严格更小的位置）
    left := make([]int, n)
    stack := []int{}
    for i := 0; i < n; i++ {
        for len(stack) > 0 && strength[stack[len(stack)-1]] >= strength[i] {
            stack = stack[:len(stack)-1]
        }
        left[i] = -1
        if len(stack) > 0 {
            left[i] = stack[len(stack)-1]
        }
        stack = append(stack, i)
    }

    // 单调栈：右边界 R（后一个更小或相等的位置）
    right := make([]int, n)
    stack = []int{}
    for i := n - 1; i >= 0; i-- {
        for len(stack) > 0 && strength[stack[len(stack)-1]] > strength[i] {
            stack = stack[:len(stack)-1]
        }
        right[i] = n
        if len(stack) > 0 {
            right[i] = stack[len(stack)-1]
        }
        stack = append(stack, i)
    }

    // 计算每个元素的贡献
    result := int64(0)
    for i := 0; i < n; i++ {
        L := left[i]
        R := right[i]
        v := int64(strength[i])

        // Σ prefix[i+1..R] = pprefix[R+1] - pprefix[i+1]
        s1 := (pprefix[R+1] - pprefix[i+1] + mod) % mod
        // Σ prefix[L+1..i] = pprefix[i+1] - pprefix[L+1]
        s2 := (pprefix[i+1] - pprefix[L+1] + mod) % mod

        partA := int64(i-L) % mod * s1 % mod
        partB := int64(R-i) % mod * s2 % mod

        contribution := v % mod * ((partA - partB + mod) % mod) % mod
        result = (result + contribution) % mod
    }

    return int(result)
}
```

**复杂度：** 时间 O(n)，空间 O(n)

以 `strength = [1,3,1,2]` 为例：

| 1 | 3 | 0 | 2 | l∈[1,1], r∈[1,1] → {[3]} | 3×3=9 |
