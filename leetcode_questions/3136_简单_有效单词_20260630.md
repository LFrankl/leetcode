## 题目描述

<p><strong>有效单词</strong> 需要满足以下几个条件：</p>

<ul>
	<li><strong>至少 </strong>包含 3 个字符。</li>
	<li>由数字 0-9 和英文大小写字母组成。（不必包含所有这类字符。）</li>
	<li><strong>至少</strong> 包含一个 <strong>元音字母 </strong>。</li>
	<li><strong>至少</strong> 包含一个 <strong>辅音字母 </strong>。</li>
</ul>

<p>给你一个字符串 <code>word</code> 。如果 <code>word</code> 是一个有效单词，则返回 <code>true</code> ，否则返回 <code>false</code> 。</p>

<p><strong>注意：</strong></p>

<ul>
	<li><code>'a'</code>、<code>'e'</code>、<code>'i'</code>、<code>'o'</code>、<code>'u'</code> 及其大写形式都属于<strong> 元音字母 </strong>。</li>
	<li>英文中的 <strong>辅音字母 </strong>是指那些除元音字母之外的字母。</li>
</ul>

<p>&nbsp;</p>

<p>这个单词满足所有条件。</p>

<p><strong>解释：</strong></p>

<p>这个单词的长度少于 3 且没有包含元音字母。</p>
</div>

<div class="example-block">

<p><strong>解释：</strong></p>

<p>这个单词包含了 <code>'$'</code> 字符且没有包含辅音字母。</p>
</div>

## AI 解答 (DeepSeek)

### 解法一：一次遍历逐项检查

思路：对字符串进行一次遍历，同时检查所有条件——长度≥3、每个字符都是数字或字母、存在至少一个元音字母、存在至少一个辅音字母。遍历时分别用两个布尔标记记录是否出现过元音和辅音。

```go
func isValidWord(word string) bool {
    if len(word) < 3 {
        return false
    }
    vowels := map[byte]bool{
        'a': true, 'e': true, 'i': true, 'o': true, 'u': true,
        'A': true, 'E': true, 'I': true, 'O': true, 'U': true,
    }
    hasVowel, hasConsonant := false, false
    for i := 0; i < len(word); i++ {
        c := word[i]
        if !((c >= '0' && c <= '9') || (c >= 'a' && c <= 'z') || (c >= 'A' && c <= 'Z')) {
            return false // 包含非法字符
        }
        if c >= 'a' && c <= 'z' || c >= 'A' && c <= 'Z' {
            if vowels[c] {
                hasVowel = true
            } else {
                hasConsonant = true
            }
        }
    }
    return hasVowel && hasConsonant
}
```

复杂度：时间 O(n)，空间 O(1)

### 解法二：利用 Unicode 分类函数简化判断

思路：使用 Go 标准库的 `unicode` 包来判断字符类别，`IsDigit` 判断数字、`IsLetter` 判断字母，配合自定义元音集合，逻辑更清晰。

```go
func isValidWord(word string) bool {
    if len(word) < 3 {
        return false
    }
    vowels := "aeiouAEIOU"
    hasVowel, hasConsonant := false, false
    for _, r := range word {
        if !unicode.IsDigit(r) && !unicode.IsLetter(r) {
            return false
        }
        if unicode.IsLetter(r) {
            if strings.ContainsRune(vowels, r) {
                hasVowel = true
            } else {
                hasConsonant = true
            }
        }
    }
    return hasVowel && hasConsonant
}
```

复杂度：时间 O(n)，空间 O(1)
