## 题目描述

「以扣会友」线下活动所在场地由若干主题空间与走廊组成，场地的地图记作由一维字符串型数组 `grid`，字符串中仅包含 `"0"～"5"` 这 6 个字符。地图上每一个字符代表面积为 1 的区域，其中 `"0"` 表示走廊，其他字符表示主题空间。相同且连续（连续指上、下、左、右四个方向连接）的字符组成同一个主题空间。

假如整个 `grid` 区域的外侧均为走廊。请问，不与走廊直接相邻的主题空间的最大面积是多少？如果不存在这样的空间请返回 `0`。

**示例 1：**
>输入：`grid = ["110","231","221"]`
>
>输出：`1`
>
>解释：4 个主题空间中，只有 1 个不与走廊相邻，面积为 1。
>![image.png](https://pic.leetcode.cn/1613708145-rscctN-image.png)


**示例 2：**
>输入：`grid = ["11111100000","21243101111","21224101221","11111101111"]`
>
>输出：`3`
>
>解释：8 个主题空间中，有 5 个不与走廊相邻，面积分别为 3、1、1、1、2，最大面积为 3。
>![image.png](https://pic.leetcode.cn/1613707985-KJyiXJ-image.png)


**提示：**
- `1 <= grid.length <= 500`
- `1 <= grid[i].length <= 500`
- `grid[i][j]` 仅可能为 `"0"～"5"`

## AI 解答 (DeepSeek)

### 分析

这道题的核心是：找到所有不与走廊（`'0'` 或网格边界外）相邻的主题空间，返回其中最大面积。

关键点：
- 相同字符且四方向连通的区域构成一个主题空间
- 网格外侧视为走廊，所以**触及网格边界的空间也算与走廊相邻**
- 需要判断每个空间是否有格子与 `'0'` 相邻或在边界上

---

### 解法一：BFS 分组 + 边界检测

思路：对每个未访问的非 `'0'` 格子做 BFS，沿相同字符四方向扩展，收集该空间的面积。在 BFS 过程中，如果发现某个格子位于网格边界，或其邻居中有 `'0'`，则标记该空间与走廊相邻。最终只统计不与走廊相邻的空间面积，取最大值。

```go
func largestArea(grid []string) int {
    n, m := len(grid), len(grid[0])
    visited := make([][]bool, n)
    for i := range visited {
        visited[i] = make([]bool, m)
    }
    
    dirs := [][2]int{{-1,0},{1,0},{0,-1},{0,1}}
    ans := 0
    
    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            if grid[i][j] == '0' || visited[i][j] {
                continue
            }
            // BFS 找同字符连通区域
            ch := grid[i][j]
            queue := [][2]int{{i, j}}
            visited[i][j] = true
            area := 0
            touchCorridor := false
            
            // 检查起始格子是否触及走廊
            if i == 0 || i == n-1 || j == 0 || j == m-1 {
                touchCorridor = true
            }
            for _, d := range dirs {
                ni, nj := i+d[0], j+d[1]
                if ni < 0 || ni >= n || nj < 0 || nj >= m || grid[ni][nj] == '0' {
                    touchCorridor = true
                }
            }
            
            for len(queue) > 0 {
                cur := queue[0]
                queue = queue[1:]
                ci, cj := cur[0], cur[1]
                area++
                
                for _, d := range dirs {
                    ni, nj := ci+d[0], cj+d[1]
                    if ni >= 0 && ni < n && nj >= 0 && nj < m {
                        if grid[ni][nj] == ch && !visited[ni][nj] {
                            visited[ni][nj] = true
                            queue = append(queue, [][2]int{{ni, nj}}...)
                            // 检查新加入格子是否触及走廊
                            if ni == 0 || ni == n-1 || nj == 0 || nj == m-1 {
                                touchCorridor = true
                            }
                            // 检查新格子的邻居是否有 '0'
                            for _, dd := range dirs {
                                nni, nnj := ni+dd[0], nj+dd[1]
                                if nni < 0 || nni >= n || nnj < 0 || nnj >= m || grid[nni][nnj] == '0' {
                                    touchCorridor = true
                                }
                            }
                        }
                    }
                }
            }
            
            if !touchCorridor && area > ans {
                ans = area
            }
        }
    }
    return ans
}
```

复杂度：时间 O(n×m)，空间 O(n×m)

### 解法二：DFS 递归 + 标记法

思路：与解法一逻辑相同，用 DFS 递归替代 BFS，代码更简洁。DFS 时若发现边界或 `'0'` 邻居，通过返回值标记该空间与走廊相邻。

```go
func largestArea(grid []string) int {
    n, m := len(grid), len(grid[0])
    visited := make([][]bool, n)
    for i := range visited {
        visited[i] = make([]bool, m)
    }
    
    dirs := [4][2]int{{-1,0},{1,0},{0,-1},{0,1}}
    ans := 0
    
    var dfs func(i, j int, ch byte) (int, bool)
    dfs = func(i, j int, ch byte) (int, bool) {
        if i < 0 || i >= n || j < 0 || j >= m {
            // 超出边界 → 邻接走廊
            return 0, true
        }
        if grid[i][j] == '0' {
            // 邻接走廊
            return 0, true
        }
        if grid[i][j] != ch || visited[i][j] {
            return 0, false
        }
        visited[i][j] = true
        area := 1
        touch := false
        for _, d := range dirs {
            a, t := dfs(i+d[0], j+d[1], ch)
            area += a
            if t {
                touch = true
            }
        }
        return area, touch
    }
    
    for i := 0; i < n; i++ {
        for j := 0; j < m; j++ {
            if grid[i][j] != '0' && !visited[i][j] {
                area, touch := dfs(i, j, grid[i][j])
                if !touch && area > ans {
                    ans = area
                }
            }
        }
    }
    return ans
}
```

复杂度：时间 O(n×m)，空间 O(n×m)（visited 数组 + DFS 递栈最深 O(n×m))

---

**小结**：两种解法本质相同——遍历每个非走廊连通区域，检测是否触及走廊（边界外或 `'0'` 邻居），只统计"封闭空间"的最大面积。DFS 写法更简洁，BFS 在极端大网格下避免递栈溢出更安全。
