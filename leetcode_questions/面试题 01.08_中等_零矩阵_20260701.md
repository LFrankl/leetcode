## 题目描述

<p>编写一种算法，若M × N矩阵中某个元素为0，则将其所在的行与列清零。</p>

<p>&nbsp;</p>

<p><strong>示例 1：</strong></p>

<pre><strong>输入：</strong>
[
  [1,1,1],
  [1,0,1],
  [1,1,1]
]
<strong>输出：</strong>
[
  [1,0,1],
  [0,0,0],
  [1,0,1]
]
</pre>

<p><strong>示例 2：</strong></p>

<pre><strong>输入：</strong>
[
  [0,1,2,0],
  [3,4,5,2],
  [1,3,1,5]
]
<strong>输出：</strong>
[
  [0,0,0,0],
  [0,4,5,0],
  [0,3,1,0]
]
</pre>

## 提示

1. 如果你在找到0时清除了行和列，则可能会清理整个矩阵。在对矩阵进行任何更改之前，首先尝试找到所有的0。

## AI 解答 (DeepSeek)

### 解法一：标记数组法

思路：先用两个数组分别记录哪些行和哪些列包含0，然后根据标记结果对对应行列全部置零。这样可以避免边遍历边置零导致新产生的0影响后续判断。

```go
func setZeroes(matrix [][]int) {
    m, n := len(matrix), len(matrix[0])
    rowZero := make([]bool, m)
    colZero := make([]bool, n)

    // 第一次遍历：标记含0的行和列
    for i := 0; i < m; i++ {
        for j := 0; j < n; j++ {
            if matrix[i][j] == 0 {
                rowZero[i] = true
                colZero[j] = true
            }
        }
    }

    // 第二次遍历：根据标记置零
    for i := 0; i < m; i++ {
        for j := 0; j < n; j++ {
            if rowZero[i] || colZero[j] {
                matrix[i][j] = 0
            }
        }
    }
}
```

复杂度：时间O(M×N)，空间O(M+N)

### 解法二：原地标记法（最优）

思路：利用矩阵的第一行和第一列本身作为标记数组，用两个额外变量记录第一行和第一列自身是否需要置零。先扫描标记，再从第二行第二列起根据标记置零，最后处理第一行和第一列，实现O(1)额外空间。

```go
func setZeroes(matrix [][]int) {
    m, n := len(matrix), len(matrix[0])
    firstRowZero, firstColZero := false, false

    // 检查第一行是否含0
    for j := 0; j < n; j++ {
        if matrix[0][j] == 0 {
            firstRowZero = true
            break
        }
    }
    // 检查第一列是否含0
    for i := 0; i < m; i++ {
        if matrix[i][0] == 0 {
            firstColZero = true
            break
        }
    }

    // 用第一行/第一列标记其余行列是否含0
    for i := 1; i < m; i++ {
        for j := 1; j < n; j++ {
            if matrix[i][j] == 0 {
                matrix[i][0] = 0 // 标记第i行
                matrix[0][j] = 0 // 标记第j列
            }
        }
    }

    // 根据标记置零（先处理非首行首列部分）
    for i := 1; i < m; i++ {
        if matrix[i][0] == 0 {
            for j := 1; j < n; j++ {
                matrix[i][j] = 0
            }
        }
    }
    for j := 1; j < n; j++ {
        if matrix[0][j] == 0 {
            for i := 1; i < m; i++ {
                matrix[i][j] = 0
            }
        }
    }

    // 最后处理第一行和第一列
    if firstRowZero {
        for j := 0; j < n; j++ {
            matrix[0][j] = 0
        }
    }
    if firstColZero {
        for i := 0; i < m; i++ {
            matrix[i][0] = 0
        }
    }
}
```

复杂度：时间O(M×N)，空间O(1)

> **关键点**：处理顺序很重要——必须先用第一行/列做标记，再根据标记置零内部元素，最后才处理第一行/列本身。否则第一行/列的标记值被提前覆盖，会导致错误。
