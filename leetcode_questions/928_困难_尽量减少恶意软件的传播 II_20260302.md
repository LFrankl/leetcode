## 题目描述

<p>给定一个由 <code>n</code> 个节点组成的网络，用 <code>n x n</code> 个邻接矩阵&nbsp;<code>graph</code>&nbsp;表示。在节点网络中，只有当&nbsp;<code>graph[i][j] = 1</code>&nbsp;时，节点&nbsp;<code>i</code>&nbsp;能够直接连接到另一个节点&nbsp;<code>j</code>。</p>

<p>一些节点&nbsp;<code>initial</code>&nbsp;最初被恶意软件感染。只要两个节点直接连接，且其中至少一个节点受到恶意软件的感染，那么两个节点都将被恶意软件感染。这种恶意软件的传播将继续，直到没有更多的节点可以被这种方式感染。</p>

<p>假设&nbsp;<code>M(initial)</code>&nbsp;是在恶意软件停止传播之后，整个网络中感染恶意软件的最终节点数。</p>

<p>我们可以从 <code>initial</code> 中&nbsp;<strong>删除一个节点</strong>，<strong>并完全移除该节点以及从该节点到任何其他节点的任何连接。</strong></p>

<p>请返回移除后能够使&nbsp;<code>M(initial)</code>&nbsp;最小化的节点。如果有多个节点满足条件，返回索引 <strong>最小的节点</strong> 。</p>

## 题目分析

这是一个关于网络感染传播的问题。给定一个无向图（邻接矩阵表示）和初始感染节点列表 `initial`。感染规则是：如果两个节点直接相连，且至少一个被感染，那么两个节点都会被感染。我们可以从 `initial` 中删除一个节点（移除该节点及其所有连接），目标是使最终感染的节点数 `M(initial)` 最小化。

**关键点**：
- 删除一个初始感染节点会移除该节点及其所有边
- 需要计算删除每个初始感染节点后，最终感染的节点数
- 选择使最终感染数最小的节点，如果有多个，返回索引最小的

## 解题思路

### 核心观察
1. 删除一个初始感染节点不仅减少了感染源，还切断了该节点与其他节点的连接
2. 感染传播过程本质上是 BFS/DFS 遍历
3. 一个节点可能被多个初始感染节点感染，删除某个感染节点可能无法拯救该节点（如果它还能被其他感染节点感染）

### 关键概念
- **感染源**：初始感染节点
- **拯救的节点数**：删除某个初始感染节点后，不会感染的节点数量
- **唯一感染源**：如果一个连通分量中只有一个初始感染节点，那么删除该节点可以拯救整个连通分量

## 解法一：BFS/DFS + 感染源统计

### 思路
1. 对于每个初始感染节点，模拟删除它（从图中移除）
2. 从剩余的初始感染节点开始 BFS/DFS 传播感染
3. 统计最终感染节点数
4. 选择使感染数最小的节点

### 复杂度分析
- 时间复杂度：O(k × n²)，其中 k = len(initial)，n 为节点数
- 空间复杂度：O(n²) 用于存储图

### 代码实现

```go
func minMalwareSpread(graph [][]int, initial []int) int {
    n := len(graph)
    sort.Ints(initial)
    
    minInfected := n + 1
    result := initial[0]
    
    // 尝试删除每个初始感染节点
    for _, removeNode := range initial {
        // 创建感染标记数组
        infected := make([]bool, n)
        
        // 初始化队列
        queue := make([]int, 0)
        for _, node := range initial {
            if node != removeNode {
                infected[node] = true
                queue = append(queue, node)
            }
        }
        
        // BFS 传播感染
        for len(queue) > 0 {
            node := queue[0]
            queue = queue[1:]
            
            for neighbor := 0; neighbor < n; neighbor++ {
                if graph[node][neighbor] == 1 && !infected[neighbor] {
                    infected[neighbor] = true
                    queue = append(queue, neighbor)
                }
            }
        }
        
        // 统计感染节点数
        count := 0
        for i := 0; i < n; i++ {
            if infected[i] {
                count++
            }
        }
        
        // 更新最小感染数
        if count < minInfected {
            minInfected = count
            result = removeNode
        }
    }
    
    return result
}
```

class Solution {
public:
    int minMalwareSpread(vector<vector<int>>& graph, vector<int>& initial) {
        int n = graph.size();
        sort(initial.begin(), initial.end());
        
        int minInfected = n + 1;
        int result = initial[0];
        
        // 尝试删除每个初始感染节点
        for (int removeNode : initial) {
            vector<bool> infected(n, false);
            queue<int> q;
            
            // 初始化感染节点（除了被删除的）
            for (int node : initial) {
                if (node != removeNode) {
                    infected[node] = true;
                    q.push(node);
                }
            }
            
            // BFS 传播感染
            while (!q.empty()) {
                int node = q.front();
                q.pop();
                
                for (int neighbor = 0; neighbor < n; neighbor++) {
                    if (graph[node][neighbor] == 1 && !infected[neighbor]) {
                        infected[neighbor] = true;
                        q.push(neighbor);
                    }
                }
            }
            
            // 统计感染节点数
            int count = 0;
            for (bool inf : infected) {
                if (inf) count++;
            }
            
            // 更新结果
            if (count < minInfected) {
                minInfected = count;
                result = removeNode;
            }
        }
        
        return result;
    }
};

## 解法二：连通分量分析 + 感染源计数

### 思路
1. 从图中移除所有初始感染节点，得到多个连通分量
2. 对于每个连通分量，统计有多少个初始感染节点与之相邻
3. 如果一个连通分量只与一个初始感染节点相邻，那么删除该感染节点可以拯救整个连通分量
4. 选择能拯救最多节点的感染节点

### 详细步骤
1. 创建新图，移除所有初始感染节点
2. 对剩余节点进行 DFS/BFS 找到连通分量
3. 对于每个连通分量，找出与之相邻的初始感染节点
4. 如果只有一个相邻的初始感染节点，该节点可以拯救这个连通分量
5. 统计每个初始感染节点能拯救的节点数
6. 选择能拯救最多节点的感染节点

### 复杂度分析
- 时间复杂度：O(n²)
- 空间复杂度：O(n²)

### 代码实现

```go
func minMalwareSpread(graph [][]int, initial []int) int {
    n := len(graph)
    sort.Ints(initial)
    
    // 将初始感染节点放入集合方便查找
    initialSet := make(map[int]bool)
    for _, node := range initial {
        initialSet[node] = true
    }
    
    // 标记数组
    visited := make([]bool, n)
    nodeToComp := make([]int, n) // 节点所属的连通分量编号
    for i := range nodeToComp {
        nodeToComp[i] = -1
    }
    
    compSize := make([]int, 0)    // 每个连通分量的大小
    compInfectedBy := make([][]int, 0) // 每个连通分量相邻的感染节点
    
    // DFS 遍历非感染节点
    var dfs func(int, int)
    dfs = func(node, compId int) {
        visited[node] = true
        nodeToComp[node] = compId
        compSize[compId]++
        
        for neighbor := 0; neighbor < n; neighbor++ {
            if graph[node][neighbor] == 1 && !visited[neighbor] && !initialSet[neighbor] {
                dfs(neighbor, compId)
            }
        }
    }
    
    // 找到所有连通分量
    compId := 0
    for node := 0; node < n; node++ {
        if !initialSet[node] && !visited[node] {
            compSize = append(compSize, 0)
            compInfectedBy = append(compInfectedBy, make([]int, 0))
            dfs(node, compId)
            compId++
        }
    }
    
    // 对于每个连通分量，找出相邻的感染节点
    for node := 0; node < n; node++ {
        if !initialSet[node] {
            compId := nodeToComp[node]
            // 检查所有邻居
            for neighbor := 0; neighbor < n; neighbor++ {
                if graph[node][neighbor] == 1 && initialSet[neighbor] {
                    // 添加感染节点到列表（去重）
                    found := false
                    for _, infNode := range compInfectedBy[compId] {
                        if infNode == neighbor {
                            found = true
                            break
                        }
                    }
                    if !found {
                        compInfectedBy[compId] = append(compInfectedBy[compId], neighbor)
                    }
                }
            }
        }
    }
    
    // 统计每个感染节点能拯救的节点数
    saveCount := make(map[int]int)
    for i := 0; i < len(compSize); i++ {
        // 如果连通分量只被一个感染节点感染
        if len(compInfectedBy[i]) == 1 {
            infectedNode := compInfectedBy[i][0]
            saveCount[infectedNode] += compSize[i]
        }
    }
    
    // 找到能拯救最多节点的感染节点
    result := initial[0]
    maxSave := -1
    for _, node := range initial {
        if saveCount[node] > maxSave {
            maxSave = saveCount[node]
            result = node
        }
    }
    
    return result
}
```

class Solution {
public:
    int minMalwareSpread(vector<vector<int>>& graph, vector<int>& initial) {
        int n = graph.size();
        sort(initial.begin(), initial.end());
        
        unordered_set<int> initialSet(initial.begin(), initial.end());
        vector<bool> visited(n, false);
        vector<int> nodeToComp(n, -1);
        
        vector<int> compSize;
        vector<unordered_set<int>> compInfectedBy;
        
        // DFS 函数
        function<void(int, int)> dfs = [&](int node, int compId) {
            visited[node] = true;
            nodeToComp[node] = compId;
            compSize[compId]++;
            
            for (int neighbor = 0; neighbor < n; neighbor++) {
                if (graph[node][neighbor] == 1 && !visited[neighbor] && 
                    !initialSet.count(neighbor)) {
                    dfs(neighbor, compId);
                }
            }
        };
        
        // 找到所有连通分量
        int compId = 0;
        for (int node = 0; node < n; node++) {
            if (!initialSet.count(node) && !visited[node]) {
                compSize.push_back(0);
                compInfectedBy.push_back(unordered_set<int>());
                dfs(node, compId);
                compId++;
            }
        }
        
        // 找出每个连通分量相邻的感染节点
        for (int node = 0; node < n; node++) {
            if (!initialSet.count(node)) {
                int cid = nodeToComp[node];
                for (int neighbor = 0; neighbor < n; neighbor++) {
                    if (graph[node][neighbor] == 1 && initialSet.count(neighbor)) {
                        compInfectedBy[cid].insert(neighbor);
                    }
                }
            }
        }
        
        // 统计每个感染节点能拯救的节点数
        unordered_map<int, int> saveCount;
        for (int i = 0; i < compSize.size(); i++) {
            if (compInfectedBy[i].size() == 1) {
                int infectedNode = *compInfectedBy[i].begin();
                saveCount[infectedNode] += compSize[i];
            }
        }
        
        // 找到能拯救最多节点的感染节点
        int result = initial[0];
        int maxSave = -1;
        for (int node : initial) {
            if (saveCount[node] > maxSave) {
                maxSave = saveCount[node];
                result = node;
            }
        }
        
        return result;
    }
};

## 解法三：并查集优化

### 思路
使用并查集来高效处理连通分量，其他思路与解法二相同。

### 复杂度分析
- 时间复杂度：O(n² × α(n))，其中 α(n) 是阿克曼函数的反函数
- 空间复杂度：O(n)

### 代码实现

```go
func minMalwareSpread(graph [][]int, initial []int) int {
    n := len(graph)
    sort.Ints(initial)
    
    // 并查集
    parent := make([]int, n)
    size := make([]int, n)
    for i := 0; i < n; i++ {
        parent[i] = i
        size[i] = 1
    }
    
    var find func(int) int
    find = func(x int) int {
        if parent[x] != x {
            parent[x] = find(parent[x])
        }
        return parent[x]
    }
    
    union := func(x, y int) {
        rootX, rootY := find(x), find(y)
        if rootX != rootY {
            if size[rootX] < size[rootY] {
                rootX, rootY = rootY, rootX
            }
            parent[rootY] = rootX
            size[rootX] += size[rootY]
        }
    }
    
    // 构建初始感染节点集合
    initialSet := make(map[int]bool)
    for _, node := range initial {
        initialSet[node] = true
    }
    
    // 连接所有非感染节点
    for i := 0; i < n; i++ {
        if initialSet[i] {
            continue
        }
        for j := i + 1; j < n; j++ {
            if initialSet[j] {
                continue
            }
            if graph[i][j] == 1 {
                union(i, j)
            }
        }
    }
    
    // 统计每个连通分量被哪些感染节点感染
    compInfectedBy := make(map[int]map[int]bool) // 连通分量根节点 -> 感染节点集合
    for i := 0; i < n; i++ {
        if initialSet[i] {
            continue
        }
        root := find(i)
        if _, exists := compInfectedBy[root]; !exists {
            compInfectedBy[root] = make(map[int]bool)
        }
        
        // 找出所有相邻的感染节点
        for j := 0; j < n; j++ {
            if graph[i][j] == 1 && initialSet[j] {
                compInfectedBy[root][j] = true
            }
        }
    }
    
    // 统计每个感染节点能拯救的节点数
    saveCount := make(map[int]int)
    for root, infectedNodes := range compInfectedBy {
        if len(infectedNodes) == 1 {
            for infectedNode := range infectedNodes {
                saveCount[infectedNode] += size[root]
            }
        }
    }
    
    // 找到能拯救最多节点的感染节点
    result := initial[0]
    maxSave := -1
    for _, node := range initial {
        if saveCount[node] > maxSave {
            maxSave = saveCount[node]
            result = node
        }
    }
    
    return result
}
```

```cpp
class Solution {
public:
    vector<int> parent, compSize;
    
    int find(int x) {
        if (parent[x] != x) {
            parent[x] = find(parent[x]);
        }
        return parent[x];
    }
    
    void unite(int x, int y) {
        int rootX = find(x);
        int rootY = find(y);
        if (rootX != rootY) {
            if (compSize[rootX] < compSize[rootY]) {
                swap(rootX, rootY);
            }
            parent[rootY] = rootX;
            compSize[rootX] += compSize[rootY];
        }
    }
    
    int minMalwareSpread(vector<vector<int>>& graph, vector<int>& initial) {
        int n = graph.size();
        sort(initial.begin(), initial.end());
        
        unordered_set<int> initialSet(initial.begin(), initial.end());
        
        // 初始化并查集
        parent.resize(n);
        compSize.resize(n, 1);
        for (int i = 0; i < n; i++) {
            parent[i] = i;
        }
        
        // 连接所有非感染节点
        for (int i = 0; i < n; i++) {
            if (initialSet.count(i)) continue;
            for (int j = i + 1; j < n; j++) {
                if (initialSet.count(j)) continue;
                if (graph[i][j] == 1) {
                    unite(i, j);
                }
            }
        }
        
        // 统计每个连通分量被哪些感染节点感染
        unordered_map<int, unordered_set<int>> compInfectedBy;
        for (int i = 0; i < n; i++) {
            if (initialSet.count(i)) continue;
            int root = find(i);
            
            // 找出所有相邻的感染节点
            for (int j = 0; j < n; j++) {
                if (graph[i][j] == 1 && initialSet.count(j)) {
                    compInfectedBy[root].insert(j);
                }
            }
        }
        
        // 统计每个感染节点能拯救的节点数
        unordered_map<int, int> saveCount;
        for (auto& [root, infectedNodes] : compInfectedBy) {
            if (infectedNodes.size() == 1) {
                int infectedNode = *infectedNodes.begin();
                saveCount[infectedNode] += compSize[root];
            }
        }
        
        // 找到能拯救最多节点的感染节点
        int result = initial[0];
        int maxSave = -1;
        for (int node : initial) {
            if (saveCount[node] > maxSave) {
                maxSave = saveCount[node];
                result = node;
            }
        }
        
        return result;
    }
};
```
