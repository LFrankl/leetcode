## 1️⃣ 题意理解

### 题目重述
给定一个日期字符串（格式为 `YYYY-MM-DD`），要求计算该日期的下一天日期，并以相同格式返回。

### 关键约束
- 日期格式固定为 `YYYY-MM-DD`
- 日期是有效的（无需验证输入有效性）
- 需要考虑闰年、月份天数变化、年份跨年等情况

### 边界情况
- 月末（如 2023-01-31 → 2023-02-01）
- 年末（如 2023-12-31 → 2024-01-01）
- 闰年二月（2024-02-28 → 2024-02-29，2024-02-29 → 2024-03-01）
- 平年二月（2023-02-28 → 2023-03-01）

### 输入输出格式
- 输入：字符串 `date`，格式 `YYYY-MM-DD`
- 输出：字符串，格式 `YYYY-MM-DD`，表示输入日期的下一天

## 2️⃣ 解题思路

### 解法一：日期解析 + 时间库
**核心思想**：利用 Go 语言标准库 `time` 处理日期计算，自动处理闰年、月份天数等复杂逻辑。

**解题步骤**：
1. 将输入字符串解析为 `time.Time` 对象
2. 使用 `AddDate(0, 0, 1)` 方法增加一天
3. 将结果格式化为目标字符串

**注意事项**：
- 时区处理：使用 UTC 避免时区影响
- 格式字符串必须与输入格式完全匹配

### 解法二：手动计算
**核心思想**：手动处理日期逻辑，避免依赖时间库，更底层地理解日期计算。

**解题步骤**：
1. 解析年、月、日为整数
2. 判断是否为闰年
3. 获取当前月份的天数
4. 如果当前日 < 当月天数，日加1
5. 否则日重置为1，月加1
6. 如果月 > 12，月重置为1，年加1
7. 格式化输出

**注意事项**：
- 月份天数数组需要考虑闰年二月
- 月份从1开始，数组索引从0开始

## 3️⃣ 代码实现

### 解法一：时间库法
```go
func nextDay(date string) string {
    // 解析输入日期，使用 UTC 时区避免本地时区影响
    t, _ := time.Parse("2006-01-02", date)
    // 增加一天
    t = t.AddDate(0, 0, 1)
    // 格式化为目标字符串
    return t.Format("2006-01-02")
}
```

### 解法二：手动计算法
```go
func nextDay(date string) string {
    // 解析年月日
    year, _ := strconv.Atoi(date[0:4])
    month, _ := strconv.Atoi(date[5:7])
    day, _ := strconv.Atoi(date[8:10])
    
    // 每月天数（索引0对应1月）
    monthDays := []int{31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31}
    
    // 判断闰年
    isLeapYear := (year%4 == 0 && year%100 != 0) || (year%400 == 0)
    if isLeapYear {
        monthDays[1] = 29 // 闰年二月29天
    }
    
    // 计算下一天
    if day < monthDays[month-1] {
        day++
    } else {
        day = 1
        if month == 12 {
            month = 1
            year++
        } else {
            month++
        }
    }
    
    // 格式化输出
    return fmt.Sprintf("%04d-%02d-%02d", year, month, day)
}
```

  - `AddDate` 也是常数时间操作
- 💾 **空间复杂度**：O(1)

  - 只使用了固定大小的变量和数组
