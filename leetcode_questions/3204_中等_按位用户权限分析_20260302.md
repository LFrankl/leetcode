## 题目分析

这是一个关于用户权限管理的题目，通常涉及位运算。假设题目要求如下（基于常见权限系统设计）：

**题目描述**：
给定一个整数数组 `permissions`，每个元素表示一个用户的权限集合（使用位掩码表示）。需要实现以下功能：
1. 检查某个用户是否拥有特定权限
2. 统计拥有特定权限的用户数量
3. 找出拥有所有指定权限的用户
4. 找出拥有任意指定权限的用户

**示例**：
```
permissions = [1, 3, 5, 7]  // 二进制：001, 011, 101, 111
权限定义：1(001)=读，2(010)=写，4(100)=执行
```

## 解题思路

### 核心思想
使用位运算来高效处理权限检查：
- **按位与 (&)**：检查是否拥有所有权限
- **按位或 (|)**：检查是否拥有任意权限
- **按位异或 (^)**：找出权限差异

## 解法一：基础位运算（逐个检查）

### 思路
遍历所有用户，对每个用户使用位运算检查权限条件。

### 代码实现

#### Go 实现
```go
package main

func hasPermission(userPerm, requiredPerm int) bool {
    return userPerm & requiredPerm == requiredPerm
}

func hasAnyPermission(userPerm, requiredPerm int) bool {
    return userPerm & requiredPerm != 0
}

func countUsersWithAllPerms(permissions []int, requiredPerm int) int {
    count := 0
    for _, perm := range permissions {
        if hasPermission(perm, requiredPerm) {
            count++
        }
    }
    return count
}

func countUsersWithAnyPerms(permissions []int, requiredPerm int) int {
    count := 0
    for _, perm := range permissions {
        if hasAnyPermission(perm, requiredPerm) {
            count++
        }
    }
    return count
}

func getUsersWithAllPerms(permissions []int, requiredPerm int) []int {
    result := []int{}
    for i, perm := range permissions {
        if hasPermission(perm, requiredPerm) {
            result = append(result, i)
        }
    }
    return result
}
```

#### C++ 实现
```cpp
#include <vector>
using namespace std;

class PermissionSystem {
public:
    bool hasPermission(int userPerm, int requiredPerm) {
        return (userPerm & requiredPerm) == requiredPerm;
    }
    
    bool hasAnyPermission(int userPerm, int requiredPerm) {
        return (userPerm & requiredPerm) != 0;
    }
    
    int countUsersWithAllPerms(vector<int>& permissions, int requiredPerm) {
        int count = 0;
        for (int perm : permissions) {
            if (hasPermission(perm, requiredPerm)) {
                count++;
            }
        }
        return count;
    }
    
    int countUsersWithAnyPerms(vector<int>& permissions, int requiredPerm) {
        int count = 0;
        for (int perm : permissions) {
            if (hasAnyPermission(perm, requiredPerm)) {
                count++;
            }
        }
        return count;
    }
    
    vector<int> getUsersWithAllPerms(vector<int>& permissions, int requiredPerm) {
        vector<int> result;
        for (int i = 0; i < permissions.size(); i++) {
            if (hasPermission(permissions[i], requiredPerm)) {
                result.push_back(i);
            }
        }
        return result;
    }
};
```

### 复杂度分析
- **时间复杂度**：O(n)，n 为用户数量
- **空间复杂度**：O(1)（统计时），O(k)（返回用户列表时，k 为符合条件的用户数）

## 解法二：位运算优化（批量处理）

### 思路
如果权限位较少（如 ≤ 32），可以预先计算每个权限位的用户集合，实现 O(1) 查询。

### 代码实现

#### Go 实现
```go
package main

type PermissionOptimizer struct {
    // bitUsers[i] 表示拥有第 i 位权限的用户索引集合
    bitUsers [32][]int
}

func NewPermissionOptimizer(permissions []int) *PermissionOptimizer {
    po := &PermissionOptimizer{}
    for userIdx, perm := range permissions {
        for bit := 0; bit < 32; bit++ {
            if perm & (1 << bit) != 0 {
                po.bitUsers[bit] = append(po.bitUsers[bit], userIdx)
            }
        }
    }
    return po
}

// 获取拥有所有指定权限的用户
func (po *PermissionOptimizer) getUsersWithAllPerms(requiredPerm int) []int {
    if requiredPerm == 0 {
        return []int{} // 空权限
    }
    
    // 找到第一个权限位的用户作为初始集合
    firstBit := 0
    for ; firstBit < 32; firstBit++ {
        if requiredPerm & (1 << firstBit) != 0 {
            break
        }
    }
    
    result := po.bitUsers[firstBit]
    // 与其他权限位求交集
    for bit := firstBit + 1; bit < 32; bit++ {
        if requiredPerm & (1 << bit) == 0 {
            continue
        }
        result = intersect(result, po.bitUsers[bit])
        if len(result) == 0 {
            break
        }
    }
    return result
}

// 获取拥有任意指定权限的用户
func (po *PermissionOptimizer) getUsersWithAnyPerms(requiredPerm int) []int {
    userSet := make(map[int]bool)
    for bit := 0; bit < 32; bit++ {
        if requiredPerm & (1 << bit) == 0 {
            continue
        }
        for _, userIdx := range po.bitUsers[bit] {
            userSet[userIdx] = true
        }
    }
    
    result := make([]int, 0, len(userSet))
    for userIdx := range userSet {
        result = append(result, userIdx)
    }
    return result
}

func intersect(a, b []int) []int {
    set := make(map[int]bool)
    for _, v := range a {
        set[v] = true
    }
    
    result := []int{}
    for _, v := range b {
        if set[v] {
            result = append(result, v)
        }
    }
    return result
}
```

#### C++ 实现
```cpp
#include <vector>
#include <unordered_set>
#include <unordered_map>
using namespace std;

class PermissionOptimizer {
private:
    vector<vector<int>> bitUsers; // bitUsers[i] 存储拥有第 i 位权限的用户索引
    
public:
    PermissionOptimizer(vector<int>& permissions) {
        bitUsers.resize(32);
        for (int i = 0; i < permissions.size(); i++) {
            int perm = permissions[i];
            for (int bit = 0; bit < 32; bit++) {
                if (perm & (1 << bit)) {
                    bitUsers[bit].push_back(i);
                }
            }
        }
    }
    
    vector<int> getUsersWithAllPerms(int requiredPerm) {
        if (requiredPerm == 0) return {};
        
        // 找到第一个权限位
        int firstBit = 0;
        while (firstBit < 32 && !(requiredPerm & (1 << firstBit))) {
            firstBit++;
        }
        
        vector<int> result = bitUsers[firstBit];
        
        // 求交集
        for (int bit = firstBit + 1; bit < 32; bit++) {
            if (!(requiredPerm & (1 << bit))) continue;
            
            unordered_set<int> set(result.begin(), result.end());
            vector<int> intersection;
            for (int userIdx : bitUsers[bit]) {
                if (set.count(userIdx)) {
                    intersection.push_back(userIdx);
                }
            }
            result = intersection;
            if (result.empty()) break;
        }
        
        return result;
    }
    
    vector<int> getUsersWithAnyPerms(int requiredPerm) {
        unordered_set<int> userSet;
        for (int bit = 0; bit < 32; bit++) {
            if (!(requiredPerm & (1 << bit))) continue;
            
            for (int userIdx : bitUsers[bit]) {
                userSet.insert(userIdx);
            }
        }
        
        return vector<int>(userSet.begin(), userSet.end());
    }
};
```

### 复杂度分析
- **预处理时间复杂度**：O(32n) ≈ O(n)
- **查询时间复杂度**：
  - `getUsersWithAllPerms`：O(k + m)，k 为最小集合大小，m 为权限位数
  - `getUsersWithAnyPerms`：O(t)，t 为相关权限位的总用户数
- **空间复杂度**：O(32n) ≈ O(n)

## 解法三：位集（Bitset）优化

### 思路
使用 bitset 表示每个用户的权限，利用位运算的并行性进行批量处理。

### 代码实现

#### Go 实现
```go
package main

const MAX_PERM_BITS = 32

type BitSetPermission struct {
    // userPerms[i] 表示第 i 位权限的用户集合（bitset）
    userPerms [MAX_PERM_BITS]uint64
    userCount int
}

func NewBitSetPermission(permissions []int) *BitSetPermission {
    bsp := &BitSetPermission{userCount: len(permissions)}
    for userIdx, perm := range permissions {
        for bit := 0; bit < MAX_PERM_BITS; bit++ {
            if perm & (1 << bit) != 0 {
                bsp.userPerms[bit] |= 1 << uint64(userIdx)
            }
        }
    }
    return bsp
}

// 获取拥有所有指定权限的用户
func (bsp *BitSetPermission) getUsersWithAllPerms(requiredPerm int) []int {
    if requiredPerm == 0 {
        return []int{}
    }
    
    // 初始化全1的bitset
    var resultBits uint64 = (1 << uint64(bsp.userCount)) - 1
    
    for bit := 0; bit < MAX_PERM_BITS; bit++ {
        if requiredPerm & (1 << bit) != 0 {
            resultBits &= bsp.userPerms[bit]
            if resultBits == 0 {
                break
            }
        }
    }
    
    // 将bitset转换为用户索引列表
    return bitsToIndices(resultBits, bsp.userCount)
}

// 获取拥有任意指定权限的用户
func (bsp *BitSetPermission) getUsersWithAnyPerms(requiredPerm int) []int {
    var resultBits uint64 = 0
    
    for bit := 0; bit < MAX_PERM_BITS; bit++ {
        if requiredPerm & (1 << bit) != 0 {
            resultBits |= bsp.userPerms[bit]
        }
    }
    
    return bitsToIndices(resultBits, bsp.userCount)
}

func bitsToIndices(bits uint64, userCount int) []int {
    result := []int{}
    for i := 0; i < userCount; i++ {
        if bits & (1 << uint64(i)) != 0 {
            result = append(result, i)
        }
    }
    return result
}
```

#### C++ 实现
```cpp
#include <vector>
#include <bitset>
using namespace std;

const int MAX_USERS = 64; // 假设最多64个用户

class BitSetPermission {
private:
    bitset<MAX_USERS> permBits[32]; // permBits[i] 表示拥有第i位权限的用户集合
    int userCount;
    
public:
    BitSetPermission(vector<int>& permissions) : userCount(permissions.size()) {
        for (int i = 0; i < permissions.size(); i++) {
            int perm = permissions[i];
            for (int bit = 0; bit < 32; bit++) {
                if (perm & (1 << bit)) {
                    permBits[bit].set(i);
                }
            }
        }
    }
    
    vector<int> getUsersWithAllPerms(int requiredPerm) {
        if (requiredPerm == 0) return {};
        
        bitset<MAX_USERS> result;
        result.set(); // 初始化为全1
        
        for (int bit = 0; bit < 32; bit++) {
            if (requiredPerm & (1 << bit)) {
                result &= permBits[bit];
                if (result.none()) break;
            }
        }
        
        return bitsetToVector(result);
    }
    
    vector<int> getUsersWithAnyPerms(int requiredPerm) {
        bitset<MAX_USERS> result;
        
        for (int bit = 0; bit < 32; bit++) {
            if (requiredPerm & (1 << bit)) {
                result |= permBits[bit];
            }
        }
        
        return bitsetToVector(result);
    }
    
private:
    vector<int> bitsetToVector(const bitset<MAX_USERS>& bs) {
        vector<int> result;
        for (int i = 0; i < userCount; i++) {
            if (bs.test(i)) {
                result.push_back(i);
            }
        }
        return result;
    }
};
```

### 复杂度分析
- **预处理时间复杂度**：O(32n) ≈ O(n)
- **查询时间复杂度**：
  - `getUsersWithAllPerms`：O(32) = O(1)（位运算常数时间）
  - `getUsersWithAnyPerms`：O(32) = O(1)
- **空间复杂度**：O(32 × MAX_USERS/64) ≈ O(1)（如果用户数固定）

## 解法对比

| 特性 | 解法一：基础位运算 | 解法二：位运算优化 | 解法三：位集优化 |
|------|-------------------|-------------------|-----------------|
| **时间复杂度** | O(n) 每次查询 | 预处理 O(n)，查询 O(k) | 预处理 O(n)，查询 O(1) |
| **空间复杂度** | O(1) | O(n) | O(1)（用户数固定时） |
| **优点** | 实现简单，无需预处理 | 多次查询时效率高 | 查询速度最快，位运算高效 |
| **缺点** | 每次查询都需遍历所有用户 | 需要额外存储空间 | 用户数受 bitset 大小限制 |
| **适用场景** | 用户数少或查询少 | 权限位少，查询频繁 | 用户数 ≤ 64，查询频繁 |

## 总结

1. **解法一**适合简单场景或一次性查询
2. **解法二**适合权限系统需要频繁查询的场景
3. **解法三**在用户数有限且查询频繁时性能最优

实际选择应根据具体需求：
- 如果用户数少（≤ 100），用解法一
- 如果权限位少且查询多，用解法二
- 如果用户数 ≤ 64 且需要极致性能，用解法三
