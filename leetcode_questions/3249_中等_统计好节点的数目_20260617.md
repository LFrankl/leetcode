## 题目描述

<p>现有一棵 <strong>无向</strong> 树，树中包含 <code>n</code> 个节点，按从 <code>0</code> 到 <code>n - 1</code> 标记。树的根节点是节点 <code>0</code> 。给你一个长度为 <code>n - 1</code> 的二维整数数组 <code>edges</code>，其中 <code>edges[i] = [a<sub>i</sub>, b<sub>i</sub>]</code> 表示树中节点 <code>a<sub>i</sub></code> 与节点 <code>b<sub>i</sub></code> 之间存在一条边。</p>

<p>如果一个节点的所有子节点为根的&nbsp;<span data-keyword="subtree">子树</span>&nbsp;包含的节点数相同，则认为该节点是一个 <strong>好节点</strong>。</p>

<p>返回给定树中<strong> 好节点 </strong>的数量。</p>

<p><strong>子树</strong>&nbsp;指的是一个节点以及它所有后代节点构成的一棵树。</p>

## AI 解答 (DeepSeek)

### 解法一：DFS 递归求子树大小

思路：先从根节点 0 做 DFS，将无向边转化为有向的父子关系，同时计算每个节点的子树大小。然后遍历所有节点，判断其所有子节点的子树大小是否完全相同。叶节点（无子节点）和只有唯一子节点的节点天然满足条件，都是好节点。

```go
func countGoodNodes(edges [][]int) int {
    n := len(edges) + 1

    // 构建邻接表
    adj := make([][]int, n)
    for _, e := range edges {
        adj[e[0]] = append(adj[e[0]], e[1])
        adj[e[1]] = append(adj[e[1]], e[0])
    }

    size := make([]int, n)      // 每个节点的子树大小
    children := make([][]int, n) // 每个节点的子节点列表（有向树）

    // DFS：建立父子关系 + 计算子树大小
    var dfs func(node, parent int) int
    dfs = func(node, parent int) int {
        size[node] = 1
        for _, neighbor := range adj[node] {
            if neighbor != parent {
                children[node] = append(children[node], neighbor)
                size[node] += dfs(neighbor, node)
            }
        }
        return size[node]
    }
    dfs(0, -1)

    // 统计好节点
    count := 0
    for i := 0; i < n; i++ {
        if len(children[i]) <= 1 {
            count++ // 0或1个子节点 → 天然是好节点
            continue
        }
        first := size[children[i][0]]
        good := true
        for _, child := range children[i] {
            if size[child] != first {
                good = false
                break
            }
        }
        if good {
            count++
        }
    }
    return count
}
```

复杂度：时间 O(n)，空间 O(n)

### 解法二：BFS 建树 + 逆序计算子树大小（避免栈溢出）

思路：当 n 达到 10^5 时，递归 DFS 可能因链状树导致栈溢出。改用 BFS 先建立父子关系和遍历序，然后按逆序（从叶到根）计算子树大小，最后统计好节点。整个过程无递归，完全迭代实现。

```go
func countGoodNodes(edges [][]int) int {
    n := len(edges) + 1

    // 构建邻接表
    adj := make([][]int, n)
    for _, e := range edges {
        adj[e[0]] = append(adj[e[0]], e[1])
        adj[e[1]] = append(adj[e[1]], e[0])
    }

    parent := make([]int, n)
    children := make([][]int, n)
    order := []int{} // BFS 遍历顺序

    // BFS 建立有向树
    visited := make([]bool, n)
    visited[0] = true
    parent[0] = -1
    queue := []int{0}
    for len(queue) > 0 {
        node := queue[0]
        queue = queue[1:]
        order = append(order, node)
        for _, neighbor := range adj[node] {
            if !visited[neighbor] {
                visited[neighbor] = true
                parent[neighbor] = node
                children[node] = append(children[node], neighbor)
                queue = append(queue, neighbor)
            }
        }
    }

    // 逆序计算子树大小（先处理叶子，再处理父节点）
    size := make([]int, n)
    for i := len(order) - 1; i >= 0; i-- {
        node := order[i]
        size[node] = 1
        for _, child := range children[node] {
            size[node] += size[child]
        }
    }

    // 统计好节点
    count := 0
    for i := 0; i < n; i++ {
        if len(children[i]) <= 1 {
            count++
            continue
        }
        first := size[children[i][0]]
        good := true
        for _, child := range children[i] {
            if size[child] != first {
                good = false
                break
            }
        }
        if good {
            count++
        }
    }
    return count
}
```

复杂度：时间 O(n)，空间 O(n)

---

**关键要点总结**：

| 条件 | 是否为好节点 |
|------|------------|
| 无子节点（叶节点） | ✅ 空集条件天然满足 |
| 仅1个子节点 | ✅ 单元素集天然满足 |
| 多个子节点，所有子树大小相同 | ✅ |
| 多个子节点，子树大小不同 | ❌ |

核心就是**一次遍历求子树大小 + 一次遍历判好节点**，两次 O(n) 即可完成。
