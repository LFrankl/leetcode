# 974. 和可被 K 整除的子数组

## 题目描述

<p>给定一个整数数组 <code>nums</code>&nbsp;和一个整数 <code>k</code> ，返回其中元素之和可被 <code>k</code>&nbsp;整除的非空&nbsp;<strong>子数组</strong> 的数目。</p>

<p><strong>子数组</strong> 是数组中&nbsp;<strong>连续</strong>&nbsp;的部分。</p>

<p>&nbsp;</p>

<p><strong>示例 1：</strong></p>

<pre>
<strong>输入：</strong>nums = [4,5,0,-2,-3,1], k = 5
<strong>输出：</strong>7
<strong>解释：
</strong>有 7 个子数组满足其元素之和可被 k = 5 整除：
[4, 5, 0, -2, -3, 1], [5], [5, 0], [5, 0, -2, -3], [0], [0, -2, -3], [-2, -3]
</pre>

<p><strong>示例 2:</strong></p>

<pre>
<strong>输入:</strong> nums = [5], k = 9
<strong>输出:</strong> 0
</pre>

<p>&nbsp;</p>

<p><strong>提示:</strong></p>

<ul>
	<li><code>1 &lt;= nums.length &lt;= 3 * 10<sup>4</sup></code></li>
	<li><code>-10<sup>4</sup>&nbsp;&lt;= nums[i] &lt;= 10<sup>4</sup></code></li>
	<li><code>2 &lt;= k &lt;= 10<sup>4</sup></code></li>
</ul>

## AI 解答 (DeepSeek)

### 解法一：前缀和 + 同余定理

思路：利用同余定理——若两个前缀和对 k 取余结果相同，则它们之间的子数组和必能被 k 整除。遍历数组计算前缀和取余，用哈希表统计每个余数出现的次数，每遇到一个余数，答案累加该余数之前出现的次数。注意负数取余需要修正为非负数。

```go
func subarraysDivByK(nums []int, k int) int {
    count := make(map[int]int)
    count[0] = 1 // 前缀和本身就能被 k 整除的情况
    prefixMod, ans := 0, 0
    for _, num := range nums {
        prefixMod = (prefixMod + num%k + k) % k // 修正负数取余
        ans += count[prefixMod]
        count[prefixMod]++
    }
    return ans
}
```

复杂度：时间 O(n)，空间 O(k)

### 解法二：前缀和 + 数组代替哈希表（常量优化版）

思路：与解法一完全相同，但由于余数范围固定为 `[0, k-1]`，可以用长度为 k 的数组替代哈希表，减少哈希开销，实测运行更快。

```go
func subarraysDivByK(nums []int, k int) int {
    modCount := make([]int, k)
    modCount[0] = 1
    prefixMod, ans := 0, 0
    for _, num := range nums {
        prefixMod = (prefixMod + num%k + k) % k
        ans += modCount[prefixMod]
        modCount[prefixMod]++
    }
    return ans
}
```

复杂度：时间 O(n)，空间 O(k)

> **核心原理图示**（以示例1为例）：
> 
> | 下标 i | num | 前缀和 | 前缀和%5 | 当前贡献 | ans |
> |--------|-----|--------|----------|----------|-----|
> | 0 | 4 | 4 | 4 | 0 | 0 |
> | 1 | 5 | 9 | 4 | 1 | 1 |
> | 2 | 0 | 9 | 4 | 2 | 3 |
> | 3 | -2 | 7 | 2 | 0 | 3 |
> | 4 | -3 | 4 | 4 | 3 | 6 |
> | 5 | 1 | 5 | 0 | 1 | **7** |
>
> 余数 4 出现了 4 次，任意两次之间构成合法子数组，共 C(4,2)=6 个；余数 0 出现 2 次，共 C(2,1)=1 个（从开头到当前位置），合计 7。
