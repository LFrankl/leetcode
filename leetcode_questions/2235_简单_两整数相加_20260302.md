## 2. 不同解法及实现

### 解法一：直接使用加法运算符（最简单直接）
这是最直观的解法，直接使用编程语言提供的加法运算符。

**时间复杂度：** O(1)  
**空间复杂度：** O(1)

```go
// Go 实现
func sum(num1 int, num2 int) int {
    return num1 + num2
}
```

```cpp
// C++ 实现
class Solution {
public:
    int sum(int num1, int num2) {
        return num1 + num2;
    }
};
```

### 解法二：使用位运算模拟加法
不使用加法运算符，通过位运算模拟加法过程。这种方法展示了计算机底层如何实现加法。

**原理：**
1. 使用异或运算得到不进位的加法结果
2. 使用与运算和左移得到进位
3. 重复直到进位为0

**时间复杂度：** O(1)（因为整数位数固定）  
**空间复杂度：** O(1)

```go
// Go 实现
func sum(num1 int, num2 int) int {
    for num2 != 0 {
        // 计算进位
        carry := num1 & num2 << 1
        // 计算不进位的和
        num1 = num1 ^ num2
        // 将进位赋值给num2，继续下一轮计算
        num2 = carry
    }
    return num1
}
```

```cpp
// C++ 实现
class Solution {
public:
    int sum(int num1, int num2) {
        while (num2 != 0) {
            // 计算进位
            unsigned int carry = (unsigned int)(num1 & num2) << 1;
            // 计算不进位的和
            num1 = num1 ^ num2;
            // 将进位赋值给num2，继续下一轮计算
            num2 = (int)carry;
        }
        return num1;
    }
};
```

### 解法三：使用数学函数（仅作展示）
使用数学库中的函数，这种方法在实际中不常用，但展示了另一种思路。

**时间复杂度：** O(1)  
**空间复杂度：** O(1)

```go
// Go 实现
import "math"

func sum(num1 int, num2 int) int {
    // 使用对数恒等式：a + b = log(e^a * e^b)
    // 注意：这种方法有精度问题，仅作展示
    if num1 == 0 {
        return num2
    }
    if num2 == 0 {
        return num1
    }
    return int(math.Log(math.Exp(float64(num1)) * math.Exp(float64(num2))))
}
```

```cpp
// C++ 实现
#include <cmath>

class Solution {
public:
    int sum(int num1, int num2) {
        // 使用对数恒等式：a + b = log(e^a * e^b)
        // 注意：这种方法有精度问题，仅作展示
        if (num1 == 0) return num2;
        if (num2 == 0) return num1;
        return static_cast<int>(log(exp(num1) * exp(num2)));
    }
};
```

### 解法四：递归实现
使用递归方式实现加法，展示函数式编程思想。

**时间复杂度：** O(|num2|)  
**空间复杂度：** O(|num2|)（递归调用栈）

```go
// Go 实现
func sum(num1 int, num2 int) int {
    // 基本情况
    if num2 == 0 {
        return num1
    }
    
    // 递归情况
    if num2 > 0 {
        return sum(num1+1, num2-1)
    } else {
        return sum(num1-1, num2+1)
    }
}
```

```cpp
// C++ 实现
class Solution {
public:
    int sum(int num1, int num2) {
        // 基本情况
        if (num2 == 0) {
            return num1;
        }
        
        // 递归情况
        if (num2 > 0) {
            return sum(num1 + 1, num2 - 1);
        } else {
            return sum(num1 - 1, num2 + 1);
        }
    }
};
```
