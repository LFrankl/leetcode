## 题目描述

<p>给你一个整数数组&nbsp;<code>nums</code>&nbsp;，如果&nbsp;<code>nums</code>&nbsp;<strong>至少</strong>&nbsp;包含 <code>2</code>&nbsp;个元素，你可以执行以下操作中的&nbsp;<strong>任意</strong>&nbsp;一个：</p>

<ul>
	<li>选择 <code>nums</code>&nbsp;中最前面两个元素并且删除它们。</li>
	<li>选择 <code>nums</code>&nbsp;中最后两个元素并且删除它们。</li>
	<li>选择 <code>nums</code>&nbsp;中第一个和最后一个元素并且删除它们。</li>
</ul>

<p>一次操作的&nbsp;<strong>分数</strong>&nbsp;是被删除元素的和。</p>

<p>在确保<strong>&nbsp;所有操作分数相同</strong>&nbsp;的前提下，请你求出&nbsp;<strong>最多</strong>&nbsp;能进行多少次操作。</p>

<p>请你返回按照上述要求&nbsp;<strong>最多</strong>&nbsp;可以进行的操作次数。</p>

<p>&nbsp;</p>

<p><strong class="example">示例 1：</strong></p>

<pre>
<b>输入：</b>nums = [3,2,1,2,3,4]
<b>输出：</b>3
<b>解释：</b>我们执行以下操作：
- 删除前两个元素，分数为 3 + 2 = 5 ，nums = [1,2,3,4] 。
- 删除第一个元素和最后一个元素，分数为 1 + 4 = 5 ，nums = [2,3] 。
- 删除第一个元素和最后一个元素，分数为 2 + 3 = 5 ，nums = [] 。
由于 nums 为空，我们无法继续进行任何操作。
</pre>

<p><strong class="example">示例 2：</strong></p>

<pre>
<b>输入：</b>nums = [3,2,6,1,4]
<b>输出：</b>2
<b>解释：</b>我们执行以下操作：
- 删除前两个元素，分数为 3 + 2 = 5 ，nums = [6,1,4] 。
- 删除最后两个元素，分数为 1 + 4 = 5 ，nums = [6] 。
至多进行 2 次操作。
</pre>

<p>&nbsp;</p>

<p><strong>提示：</strong></p>

<ul>
	<li><code>2 &lt;= nums.length &lt;= 2000</code></li>
	<li><code>1 &lt;= nums[i] &lt;= 1000</code></li>
</ul>

## 解题分析

**题目核心**：从数组两端或两端各取两个元素删除，所有操作的分数（被删元素之和）必须相同，求最多操作次数。

**关键观察**：第一操作决定了后续所有操作的目标分数。第一操作只能是三种选择之一（前两个、后两个、首尾），因此目标分数至多只有3种可能值。确定目标分数后，问题转化为：在子数组上，用相同分数做尽可能多的操作——这是经典的区间DP。

---

### 解法一：区间DP（迭代）

思路：枚举三种可能的目标分数，对每种目标用区间DP求解。`dp[i][j]` 表示子数组 `nums[i..j]` 在目标分数 `target` 下最多能做的操作数。每次可以从左端删两个、右端删两个、或删首尾，只要和等于 target 就转移。使用 n+2 的数组做 padding，自然处理边界（剩余元素不足时 dp 值为 0）。

```go
func maxOperations(nums []int) int {
    n := len(nums)

    helper := func(target int) int {
        // dp[i][j]: 子数组 nums[i..j] 在目标分数 target 下的最大操作数
        // 额外 +2 行/列做 padding，越界访问自动返回 0
        dp := make([][]int, n+2)
        for i := range dp {
            dp[i] = make([]int, n+2)
        }
        for i := n - 1; i >= 0; i-- {
            for j := i + 1; j < n; j++ {
                res := 0
                // 删除最前面两个
                if nums[i] + nums[i+1] == target {
                    res = max(res, 1 + dp[i+2][j])
                }
                // 删除最后两个
                if nums[j-1] + nums[j] == target {
                    res = max(res, 1 + dp[i][j-2])
                }
                // 删除第一个和最后一个
                if nums[i] + nums[j] == target {
                    res = max(res, 1 + dp[i+1][j-1])
                }
                dp[i][j] = res
            }
        }
        return dp[0][n-1]
    }

    s1 := nums[0] + nums[1]
    s2 := nums[n-2] + nums[n-1]
    s3 := nums[0] + nums[n-1]

    ans := helper(s1)
    if s2 != s1 {
        ans = max(ans, helper(s2))
    }
    if s3 != s1 && s3 != s2 {
        ans = max(ans, helper(s3))
    }
    return ans
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

复杂度：时间 O(n²)（每种目标分数 O(n²)，至多3种），空间 O(n²)

### 解法二：区间DP（记忆化递归）

思路：与解法一相同的DP定义，但用递归+map记忆化实现。只计算可达状态，避免填充整个矩阵。当很多状态不可达时，实际计算量可能少于 O(n²)。

```go
func maxOperations(nums []int) int {
    n := len(nums)

    helper := func(target int) int {
        memo := make(map[int]int)
        var dfs func(i, j int) int
        dfs = func(i, j int) int {
            if i >= j {
                return 0
            }
            key := i * n + j
            if v, ok := memo[key]; ok {
                return v
            }
            res := 0
            if nums[i] + nums[i+1] == target {
                res = max(res, 1 + dfs(i+2, j))
            }
            if nums[j-1] + nums[j] == target {
                res = max(res, 1 + dfs(i, j-2))
            }
            if nums[i] + nums[j] == target {
                res = max(res, 1 + dfs(i+1, j-1))
            }
            memo[key] = res
            return res
        }
        return dfs(0, n-1)
    }

    s1 := nums[0] + nums[1]
    s2 := nums[n-2] + nums[n-1]
    s3 := nums[0] + nums[n-1]

    ans := helper(s1)
    if s2 != s1 {
        ans = max(ans, helper(s2))
    }
    if s3 != s1 && s3 != s2 {
        ans = max(ans, helper(s3))
    }
    return ans
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

复杂度：时间 O(n²)（最坏情况），空间 O(n²)（map 存储可达状态数，最坏 O(n²)，实际可能更少）

---

**总结**：解法一迭代DP更稳定、无递归栈溢出风险，推荐使用。两种方法核心思想相同——枚举目标分数 + 区间DP，是本题的最优解法。
