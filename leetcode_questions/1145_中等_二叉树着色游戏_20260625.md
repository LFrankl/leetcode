## 题目描述

<p>有两位极客玩家参与了一场「二叉树着色」的游戏。游戏中，给出二叉树的根节点&nbsp;<code>root</code>，树上总共有 <code>n</code> 个节点，且 <code>n</code> 为奇数，其中每个节点上的值从&nbsp;<code>1</code> 到&nbsp;<code>n</code>&nbsp;各不相同。</p>

<p>最开始时：</p>

<ul>
	<li>「一号」玩家从 <code>[1, n]</code>&nbsp;中取一个值&nbsp;<code>x</code>（<code>1 &lt;= x &lt;= n</code>）；</li>
	<li>「二号」玩家也从&nbsp;<code>[1, n]</code>&nbsp;中取一个值&nbsp;<code>y</code>（<code>1 &lt;= y &lt;= n</code>）且&nbsp;<code>y != x</code>。</li>
</ul>

<p>「一号」玩家给值为&nbsp;<code>x</code>&nbsp;的节点染上红色，而「二号」玩家给值为&nbsp;<code>y</code>&nbsp;的节点染上蓝色。</p>

<p>之后两位玩家轮流进行操作，「一号」玩家先手。每一回合，玩家选择一个被他染过色的节点，将所选节点一个 <strong>未着色 </strong>的邻节点（即左右子节点、或父节点）进行染色（「一号」玩家染红色，「二号」玩家染蓝色）。</p>

<p>如果（且仅在此种情况下）当前玩家无法找到这样的节点来染色时，其回合就会被跳过。</p>

<p>若两个玩家都没有可以染色的节点时，游戏结束。着色节点最多的那位玩家获得胜利 ✌️。</p>

<p>现在，假设你是「二号」玩家，根据所给出的输入，假如存在一个&nbsp;<code>y</code>&nbsp;值可以确保你赢得这场游戏，则返回&nbsp;<code>true</code> ；若无法获胜，就请返回 <code>false</code> 。</p>

## AI 解答 (DeepSeek)

## 简要思路

一号玩家选了节点 x 后，整棵树被分成三个"势力区域"：x 的左子树、x 的右子树、以及 x 父节点那一侧的剩余部分。二号玩家只需选 x 的某个邻居（左子、右子或父节点），就能"切断"一号玩家通往该区域的路径，独占该区域所有节点。若三个区域中最大的那个超过 n/2，二号玩家即可获胜。

---

### 解法一：DFS 计算子树大小 + 比较三大区域

思路：先 DFS 找到节点 x，再 DFS 计算 x 左子树和右子树的大小，父侧区域大小 = n - 1 - leftSize - rightSize。取三个区域的最大值，若 > n/2 则二号必胜。

```go
/**
 * Definition for a binary tree node.
 * type TreeNode struct {
 *     Val int
 *     Left *TreeNode
 *     Right *TreeNode
 * }
 */
func btreeGameWinningMove(root *TreeNode, n int, x int) bool {
    var target *TreeNode
    // 找到值为 x 的节点
    findNode(root, x, &target)

    leftSize  := countNodes(target.Left)
    rightSize := countNodes(target.Right)
    parentSize := n - 1 - leftSize - rightSize

    maxRegion := max(leftSize, rightSize, parentSize)
    return maxRegion > n/2
}

func findNode(node *TreeNode, x int, target **TreeNode) {
    if node == nil || *target != nil {
        return
    }
    if node.Val == x {
        *target = node
        return
    }
    findNode(node.Left, x, target)
    findNode(node.Right, x, target)
}

func countNodes(node *TreeNode) int {
    if node == nil {
        return 0
    }
    return 1 + countNodes(node.Left) + countNodes(node.Right)
}

func max(a, b, c int) int {
    if a >= b && a >= c {
        return a
    }
    if b >= a && b >= c {
        return b
    }
    return c
}
```

复杂度：时间 O(n)，空间 O(n)（递归栈深度）

---

### 解法二：一次 DFS 同时找节点并计算子树大小

思路：在同一个 DFS 中，找到节点 x 时直接返回其子树大小，避免第二次遍历。用一个全局变量记录 x 节点的左子树和右子树大小。

```go
func btreeGameWinningMove(root *TreeNode, n int, x int) bool {
    leftSize, rightSize := 0, 0
    dfs(root, x, &leftSize, &rightSize)
    parentSize := n - 1 - leftSize - rightSize
    return max3(leftSize, rightSize, parentSize) > n/2
}

// 返回以 node 为根的子树大小；当遇到 x 时，记录其左右子树大小
func dfs(node *TreeNode, x int, ls, rs *int) int {
    if node == nil {
        return 0
    }
    l := dfs(node.Left, x, ls, rs)
    r := dfs(node.Right, x, ls, rs)
    if node.Val == x {
        *ls = l
        *rs = r
    }
    return 1 + l + r
}

func max3(a, b, c int) int {
    m := a
    if b > m { m = b }
    if c > m { m = c }
    return m
}
```

复杂度：时间 O(n)，空间 O(n)（递归栈深度）

---

**总结**：核心观察是二号玩家选 x 的邻居可以"封锁"最大区域。两种解法本质相同，解法二将两次 DFS 合为一次，更简洁。
