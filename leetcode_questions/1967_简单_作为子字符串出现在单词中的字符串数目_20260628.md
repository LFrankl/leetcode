# 1967. 作为子字符串出现在单词中的字符串数目

**难度**: Easy

**标签**: 数组, 字符串

**链接**: https://leetcode.cn/problems/number-of-strings-that-appear-as-substrings-in-word/

## 题目描述

<p>给你一个字符串数组 <code>patterns</code> 和一个字符串 <code>word</code> ，统计 <code>patterns</code> 中有多少个字符串是 <code>word</code> 的子字符串。返回字符串数目。</p>

<p><strong>子字符串</strong> 是字符串中的一个连续字符序列。</p>

<p>&nbsp;</p>

<p><strong>示例 1：</strong></p>

<pre>
<strong>输入：</strong>patterns = ["a","abc","bc","d"], word = "abc"
<strong>输出：</strong>3
<strong>解释：</strong>
- "a" 是 "<em><strong>a</strong></em>bc" 的子字符串。
- "abc" 是 "<em><strong>abc</strong></em>" 的子字符串。
- "bc" 是 "a<em><strong>bc</strong></em>" 的子字符串。
- "d" 不是 "abc" 的子字符串。
patterns 中有 3 个字符串作为子字符串出现在 word 中。
</pre>

<p><strong>示例 2：</strong></p>

<pre>
<strong>输入：</strong>patterns = ["a","b","c"], word = "aaaaabbbbb"
<strong>输出：</strong>2
<strong>解释：</strong>
- "a" 是 "a<em><strong>a</strong></em>aaabbbbb" 的子字符串。
- "b" 是 "aaaaabbbb<em><strong>b</strong></em>" 的子字符串。
- "c" 不是 "aaaaabbbbb" 的字符串。
patterns 中有 2 个字符串作为子字符串出现在 word 中。
</pre>

<p><strong>示例 3：</strong></p>

<pre>
<strong>输入：</strong>patterns = ["a","a","a"], word = "ab"
<strong>输出：</strong>3
<strong>解释：</strong>patterns 中的每个字符串都作为子字符串出现在 word "<em><strong>a</strong></em>b" 中。
</pre>

<p>&nbsp;</p>

<p><strong>提示：</strong></p>

<ul>
	<li><code>1 &lt;= patterns.length &lt;= 100</code></li>
	<li><code>1 &lt;= patterns[i].length &lt;= 100</code></li>
	<li><code>1 &lt;= word.length &lt;= 100</code></li>
	<li><code>patterns[i]</code> 和 <code>word</code> 由小写英文字母组成</li>
</ul>

## AI 解答 (DeepSeek)

### 解法一：直接遍历检查子字符串

思路：对 `patterns` 中每个字符串，检查它是否是 `word` 的子字符串。大多数语言都内置了子字符串查找方法（如 `Contains`/`indexOf`），直接调用即可。遍历一遍 `patterns` 数组，统计满足条件的个数。

```go
func numOfStrings(patterns []string, word string) int {
    count := 0
    for _, p := range patterns {
        if strings.Contains(word, p) {
            count++
        }
    }
    return count
}
```

复杂度：时间 O(n × m × L)，其中 n = patterns长度，m = word长度，L = patterns[i]平均长度（`strings.Contains` 内部为 O(m×L)）；空间 O(1)

---

### 解法二：预处理 word 所有子字符串

思路：先枚举 `word` 的所有子字符串存入哈希集合，然后遍历 `patterns`，只需查集合即可判断，将每次查找从 O(m×L) 降为 O(L)（哈希查找）。当 `patterns` 较多时更高效。

```go
func numOfStrings(patterns []string, word string) int {
    // 预处理：枚举 word 的所有子字符串
    subStrSet := make(map[string]bool)
    for i := 0; i < len(word); i++ {
        for j := i + 1; j <= len(word); j++ {
            subStrSet[word[i:j]] = true
        }
    }
    // 查找
    count := 0
    for _, p := range patterns {
        if subStrSet[p] {
            count++
        }
    }
    return count
}
```

复杂度：时间 O(m² + n × L)，预处理枚举所有子字符串 O(m²)，查找 O(n×L)；空间 O(m²)，存储所有子字符串
