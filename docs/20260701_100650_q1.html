409. 最长回文串

难度: Easy

标签: 贪心, 哈希表, 字符串

题目描述

给定一个包含大写字母和小写字母的字符串 s ，返回 通过这些字母构造成的 最长的 回文串 的长度。

在构造过程中，请注意 区分大小写 。比如 "Aa" 不能当做一个回文字符串。

 

示例 1: 

输入:s = "abccccdd"
输出:7
解释:
我们可以构造的最长的回文串是"dccaccd", 它的长度是 7。

示例 2:

输入:s = "a"
输出:1
解释：可以构造的最长回文串是"a"，它的长度是 1。

 

提示:

  • 1 <= s.length <= 2000
  • s 只由小写 和/或 大写英文字母组成

AI 解答 (DeepSeek)

解法一：贪心 + 字符计数

思路：回文串左右对称，中间可放一个单独字符。统计每个字符出现次数，偶数次全部可用，奇数次取最大偶数部分（count - 1），最后若存在奇数次字符则中心再加 1。

func longestPalindrome(s string) int {
    count := make(map[rune]int)
    for _, ch := range s {
        count[ch]++
    }

    length := 0
    hasOdd := false
    for _, freq := range count {
        length += freq / 2 * 2  // 取偶数部分
        if freq % 2 == 1 {
            hasOdd = true
        }
    }
    if hasOdd {
        length += 1  // 中间放一个奇数字符
    }
    return length
}

复杂度：时间 O(n)，空间 O(k)（k 为字符种类数，最多52）

解法二：贪心 + 数组计数（最优）

思路：与解法一相同逻辑，但用固定大小数组替代哈希表，利用 ASCII 值直接映射索引，减少哈希开销。

func longestPalindrome(s string) int {
    freq := make([]int, 128)
    for _, ch := range s {
        freq[ch]++
    }

    length := 0
    for _, f := range freq {
        length += f / 2 * 2
        if f % 2 == 1 && length % 2 == 0 {
            length += 1
        }
    }
    return length
}

复杂度：时间 O(n)，空间 O(1)（固定128大小数组）

核心要点：贪心策略——偶数全用，奇数留一个作中心。解法二用 length % 2 == 0 判断是否已添加中心字符，更简洁且只需一次遍历 freq 数组。
