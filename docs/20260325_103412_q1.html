1️⃣ 题意理解

题目重述： 给定一个已按升序排列的整数数组 nums 和一个目标值 target，要求找到 target 在数组中的上界。上界定义为第一个大于 target 的元素索引。如果所有元素都不大于 target，则返回数组长度。

关键约束： - 数组已按升序排列 - 数组可能为空 - 元素可能重复 - 需要找到第一个大于 target 的位置，而不是等于

边界情况： - 空数组：返回 0 - 所有元素 ≤ target：返回数组长度 - 所有元素 > target：返回 0 - 有多个等于 target 的元素：返回第一个大于 target 的位置

输入输出： - 输入：nums []int, target int - 输出：int 类型，表示上界索引

2️⃣ 解题思路

解法一：线性扫描

核心思想：从左到右遍历数组，找到第一个大于 target 的元素。

步骤： 1. 遍历数组中的每个元素 2. 如果当前元素 > target，返回当前索引 3. 如果遍历结束都没有找到，返回数组长度

注意事项： - 需要处理空数组情况 - 注意边界条件

解法二：二分查找

核心思想：利用数组有序的特性，使用二分查找加速搜索。

步骤： 1. 初始化左右指针 left = 0, right = len(nums) 2. 当 left < right 时循环： - 计算中点 mid = left + (right - left)/2 - 如果 nums[mid] > target，说明上界在左半部分，right = mid - 否则，说明上界在右半部分，left = mid + 1 3. 返回 left 即为上界位置

注意事项： - 循环条件是 left < right 而不是 left ≤ right - 中点计算防止溢出 - 最终返回的是 left 而不是 mid

3️⃣ 代码实现

解法一：线性扫描

func upperBound(nums []int, target int) int {
    // 遍历数组
    for i, num := range nums {
        // 找到第一个大于 target 的元素
        if num > target {
            return i
        }
    }
    // 所有元素都不大于 target
    return len(nums)
}

解法二：二分查找

func upperBound(nums []int, target int) int {
    left, right := 0, len(nums)

    // 二分查找
    for left < right {
        mid := left + (right-left)/2  // 防止溢出

        if nums[mid] > target {
            // 中点大于 target，上界在左半部分（包含中点）
            right = mid
        } else {
            // 中点小于等于 target，上界在右半部分
            left = mid + 1
        }
    }

    // left 即为第一个大于 target 的位置
    return left
}

4️⃣ 复杂度分析

解法一：线性扫描

  • ⏱️ 时间复杂度：O(n)
    • 最坏情况下需要遍历整个数组
    • n 为数组长度
  • 💾 空间复杂度：O(1)
    • 只使用了常数级别的额外空间

解法二：二分查找

  • ⏱️ 时间复杂度：O(log n)
    • 每次将搜索范围减半
    • 最多需要 log₂n 次比较
  • 💾 空间复杂度：O(1)
    • 只使用了几个指针变量
