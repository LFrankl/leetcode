861. 翻转矩阵后的得分

难度: Medium

标签: 贪心, 位运算, 数组, 矩阵

链接: https://leetcode.cn/problems/score-after-flipping-matrix/

题目描述

给你一个大小为 m x n 的二元矩阵 grid ，矩阵中每个元素的值为 0 或 1 。

一次 移动 是指选择任一行或列，并转换该行或列中的每一个值：将所有 0 都更改为 1，将所有 1 都更改为 0。

在做出任意次数的移动后，将该矩阵的每一行都按照二进制数来解释，矩阵的 得分 就是这些数字的总和。

在执行任意次 移动 后（含 0 次），返回可能的最高分数。

 

示例 1：

输入：grid = [[0,0,1,1],[1,0,1,0],[1,1,0,0]]
输出：39
解释：0b1111 + 0b1001 + 0b1111 = 15 + 9 + 15 = 39

示例 2：

输入：grid = [[0]]
输出：1

 

提示：

  • m == grid.length
  • n == grid[i].length
  • 1 <= m, n <= 20
  • grid[i][j] 为 0 或 1

1️⃣ 题意理解

核心要求：
给定一个 m×n 的二进制矩阵，我们可以对任意行或列进行“翻转”操作（将 0 变 1，1 变 0），目标是使矩阵的“得分”最大化。得分定义为将每一行视为二进制数，求和得到的总和。

关键约束： - 矩阵元素仅为 0 或 1 - 可以执行任意次行/列翻转（包括 0 次） - 矩阵大小 m, n ≤ 20

边界情况： - 矩阵为 1×1 时，翻转或不翻转，最大得分为 1 - 矩阵只有一行时，只需考虑该行翻转与否 - 矩阵只有一列时，只需考虑该列翻转与否

输入输出： - 输入：二维整数数组 grid，元素为 0 或 1 - 输出：整数，表示最大得分

2️⃣ 解题思路

解法一：贪心算法（逐位最大化）

核心思想： 1. 行翻转策略：确保第一列全为 1（因为二进制数的最高位权重最大） 2. 列翻转策略：从第二列开始，每列尽可能多地让 1 的数量最多（因为列翻转影响该列所有行） 3. 贪心依据：高位权重远大于低位，优先保证高位为 1

步骤： 1. 遍历每一行，如果该行第一个元素为 0，则翻转整行 2. 遍历每一列（从第二列开始），统计该列中 1 的数量 3. 如果该列 1 的数量少于一半（即少于 m/2），则翻转该列 4. 计算最终矩阵的得分

注意事项： - 行翻转只影响当前行，列翻转影响所有行 - 列翻转时，1 的数量 = 原始数量（如果该行未翻转）或 m - 原始数量（如果该行已翻转） - 实际计算得分时，可以不用真正修改矩阵，直接按权重计算

解法二：数学优化（直接计算）

核心思想： 1. 第一列必须全为 1（通过行翻转实现） 2. 对于后续每一列，该列贡献的得分 = max(该列1的数量, m - 该列1的数量) * 该列权重 3. 权重从最高位到最低位递减：2^(n-1-j)

步骤： 1. 遍历每一行，如果 grid[i][0] == 0，则翻转该行（实际只需记录翻转状态） 2. 对于每一列 j： - 统计该列在行翻转后的 1 的数量 - 该列贡献 = max(cnt, m - cnt) * (1 << (n-1-j)) 3. 累加所有列贡献得到总分

优势： - 无需真正修改矩阵 - 一次遍历即可完成计算

3️⃣ 代码实现

解法一：贪心算法（显式翻转）

func matrixScore(grid [][]int) int {
    m, n := len(grid), len(grid[0])

    // 第一步：确保第一列全为1（通过行翻转）
    for i := 0; i < m; i++ {
        if grid[i][0] == 0 {
            // 翻转整行
            for j := 0; j < n; j++ {
                grid[i][j] ^= 1 // 0变1，1变0
            }
        }
    }

    // 第二步：从第二列开始，确保每列1的数量尽可能多
    for j := 1; j < n; j++ {
        cnt := 0
        for i := 0; i < m; i++ {
            cnt += grid[i][j]
        }
        // 如果该列1的数量少于一半，则翻转该列
        if cnt < m-cnt {
            for i := 0; i < m; i++ {
                grid[i][j] ^= 1
            }
        }
    }

    // 第三步：计算得分
    score := 0
    for i := 0; i < m; i++ {
        rowVal := 0
        for j := 0; j < n; j++ {
            rowVal = rowVal*2 + grid[i][j] // 二进制转十进制
        }
        score += rowVal
    }
    return score
}

解法二：数学优化（直接计算）

func matrixScore(grid [][]int) int {
    m, n := len(grid), len(grid[0])
    score := 0

    // 第一列权重：2^(n-1)
    // 通过行翻转，第一列必然全为1，贡献为 m * (1 << (n-1))
    score += m * (1 << (n-1))

    // 处理后续列
    for j := 1; j < n; j++ {
        cnt := 0 // 统计该列1的数量（考虑行翻转后）
        for i := 0; i < m; i++ {
            // 如果该行第一个元素是0，说明该行被翻转了
            // 因此当前元素的实际值 = grid[i][j] ^ grid[i][0]
            // 因为如果grid[i][0]原来是0，行翻转后grid[i][j]也翻转了
            // 等价于：如果grid[i][0]==1，取原值；否则取反
            if grid[i][0] == 1 {
                cnt += grid[i][j]
            } else {
                cnt += 1 - grid[i][j] // 翻转后的值
            }
        }
        // 该列贡献 = max(cnt, m-cnt) * 权重
        maxCnt := cnt
        if m-cnt > maxCnt {
            maxCnt = m - cnt
        }
        score += maxCnt * (1 << (n-1-j))
    }
    return score
}
