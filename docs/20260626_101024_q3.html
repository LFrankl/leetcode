805. 数组的均值分割

难度: Hard

标签: 位运算, 数组, 哈希表, 数学, 动态规划, 位掩码

链接: https://leetcode.cn/problems/split-array-with-same-average/

题目描述

给定你一个整数数组 nums

我们要将 nums 数组中的每个元素移动到 A 数组 或者 B 数组中，使得 A 数组和 B 数组不为空，并且 average(A) == average(B) 。

如果可以完成则返回true ， 否则返回 false  。

注意：对于数组 arr ,  average(arr) 是 arr 的所有元素的和除以 arr 长度。

 

示例 1:

输入: nums = [1,2,3,4,5,6,7,8]
输出: true
解释: 我们可以将数组分割为 [1,4,5,8] 和 [2,3,6,7], 他们的平均值都是4.5。

示例 2:

输入: nums = [3,1]
输出: false

 

提示:

  • 1 <= nums.length <= 30
  • 0 <= nums[i] <= 104

解法一：折半搜索（Meet in the Middle）

思路：关键数学观察——若 average(A) = average(B) 且 A∪B = nums，则 average(A) = average(nums)，即 sum(A)·n = totalSum·|A|。因此只需找到某个非空子集 A（1 ≤ |A| ≤ n-1）满足该等式。将数组分成两半，分别枚举所有子集的 (sum, size)，再折半组合检查是否存在满足条件的配对。

func splitArraySameAverage(nums []int) bool {
    n := len(nums)
    if n < 2 {
        return false
    }

    totalSum := 0
    for _, x := range nums {
        totalSum += x
    }

    n1 := n / 2
    left := nums[:n1]
    right := nums[n1:]
    n2 := n - n1

    // 枚举左半所有子集，按 size 分组记录 sum
    leftSums := make([][]int, n1+1)
    for mask := 0; mask < (1 << n1); mask++ {
        sum, size := 0, 0
        for i := 0; i < n1; i++ {
            if mask & (1 << i) != 0 {
                sum += left[i]
                size++
            }
        }
        leftSums[size] = append(leftSums[size], sum)
    }

    // 枚举右半所有子集，按 size 分组记录 sum
    rightSums := make([][]int, n2+1)
    for mask := 0; mask < (1 << n2); mask++ {
        sum, size := 0, 0
        for i := 0; i < n2; i++ {
            if mask & (1 << i) != 0 {
                sum += right[i]
                size++
            }
        }
        rightSums[size] = append(rightSums[size], sum)
    }

    // 对每个可能的子集大小 k，检查是否能拼出目标 sum
    for k := 1; k <= n-1; k++ {
        if totalSum * k % n != 0 {
            continue
        }
        target := totalSum * k / n

        for k1 := 0; k1 <= n1 && k1 <= k; k1++ {
            k2 := k - k1
            if k2 > n2 {
                continue
            }
            // 用 hash set 查找右半 size=k2 的 sum
            rightSet := make(map[int]bool, len(rightSums[k2]))
            for _, s := range rightSums[k2] {
                rightSet[s] = true
            }
            for _, s1 := range leftSums[k1] {
                if rightSet[target - s1] {
                    return true
                }
            }
        }
    }

    return false
}

复杂度：时间 O(n · 2^(n/2))，空间 O(2^(n/2))

解法二：动态规划（按子集大小分组求和）

思路：同样利用 sum(A)·n = totalSum·|A| 的数学结论。用 dp[k] 记录恰好选取 k 个元素时所有可达的 sum 值，逐元素更新。最终检查是否存在某个 k（1 ≤ k ≤ n-1）使得 totalSum·k/n 为整数且出现在 dp[k] 中。

func splitArraySameAverage(nums []int) bool {
    n := len(nums)
    totalSum := 0
    for _, x := range nums {
        totalSum += x
    }

    // dp[k] = 恰好选 k 个元素时可达到的所有 sum
    dp := make([]map[int]bool, n+1)
    dp[0] = map[int]bool{0: true}
    for i := 1; i <= n; i++ {
        dp[i] = map[int]bool{}
    }

    for _, x := range nums {
        // 倒序更新，避免同一元素被重复选取
        for k := n; k >= 1; k-- {
            for s := range dp[k-1] {
                dp[k][s+x] = true
            }
        }
    }

    for k := 1; k < n; k++ {
        if totalSum * k % n != 0 {
            continue
        }
        if dp[k][totalSum*k/n] {
            return true
        }
    }

    return false
}

复杂度：时间 O(n · S)（S 为所有不同 (sum, size) 状态数，上界约 O(n² · maxVal)），空间 O(S)

对比：解法一（折半搜索）将 2^30 的搜索空间降至 2^15 × 2，是 n≤30 时的最优解法；解法二（DP）思路更直观，但状态数可能较多，在最坏情况下空间开销更大。
