题目描述

给你两棵二叉树的根节点 p 和 q ，编写一个函数来检验这两棵树是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

 

示例 1：

输入：p = [1,2,3], q = [1,2,3]
输出：true

示例 2：

输入：p = [1,2], q = [1,null,2]
输出：false

示例 3：

输入：p = [1,2,1], q = [1,1,2]
输出：false

 

提示：

  • 两棵树上的节点数目都在范围 [0, 100] 内
  • -104 <= Node.val <= 104

题目分析

核心问题：判断两棵二叉树是否完全相同——结构一致且每个节点值相等。

关键点： - 不仅要值相同，结构也必须相同（空节点位置也要一致） - 两棵树都可能为空 - 需要同时遍历两棵树的对应节点进行比较

解法一：递归（DFS）

思路：自顶向下，同时遍历两棵树的对应节点，逐个比较。

  • 两个节点都为空 → 相同
  • 一个为空一个不为空 → 不同
  • 值不等 → 不同
  • 值相等 → 递归比较左子树和右子树
func isSameTree(p *TreeNode, q *TreeNode) bool {
    if p == nil && q == nil {
        return true
    }
    if p == nil || q == nil {
        return false
    }
    if p.Val != q.Val {
        return false
    }
    return isSameTree(p.Left, q.Left) && isSameTree(p.Right, q.Right)
}

class Solution {
public:
    bool isSameTree(TreeNode* p, TreeNode* q) {
        if (!p && !q) return true;
        if (!p || !q) return false;
        if (p->val != q->val) return false;
        return isSameTree(p->left, q->left) && isSameTree(p->right, q->right);
    }
};

复杂度分析： - 时间复杂度：O(min(m, n))，m、n 为两棵树节点数，只需遍历到第一个不同点即可提前返回 - 空间复杂度：O(min(h₁, h₂))，h₁、h₂ 为树高，递归栈深度取决于较矮的树

解法二：迭代（BFS，层序遍历）

思路：用两个队列同时层序遍历两棵树，逐层逐节点比较。

func isSameTree(p *TreeNode, q *TreeNode) bool {
    queue1 := []*TreeNode{p}
    queue2 := []*TreeNode{q}

    for len(queue1) > 0 && len(queue2) > 0 {
        node1 := queue1[0]
        node2 := queue2[0]
        queue1 = queue1[1:]
        queue2 = queue2[1:]

        if node1 == nil && node2 == nil {
            continue
        }
        if node1 == nil || node2 == nil {
            return false
        }
        if node1.Val != node2.Val {
            return false
        }

        queue1 = append(queue1, node1.Left, node1.Right)
        queue2 = append(queue2, node2.Left, node2.Right)
    }

    return len(queue1) == 0 && len(queue2) == 0
}

class Solution {
public:
    bool isSameTree(TreeNode* p, TreeNode* q) {
        queue<TreeNode*> q1, q2;
        q1.push(p);
        q2.push(q);

        while (!q1.empty() && !q2.empty()) {
            auto node1 = q1.front(); q1.pop();
            auto node2 = q2.front(); q2.pop();

            if (!node1 && !node2) continue;
            if (!node1 || !node2) return false;
            if (node1->val != node2->val) return false;

            q1.push(node1->left);
            q1.push(node1->right);
            q2.push(node2->left);
            q2.push(node2->right);
        }

        return q1.empty() && q2.empty();
    }
};

复杂度分析： - 时间复杂度：O(min(m, n)) - 空间复杂度：O(min(m, n))，队列最多存储一层的节点，最坏情况（完全二叉树底层）约 O(n/2)

解法三：迭代（DFS，栈模拟递归）

思路：用栈模拟递归过程，显式管理遍历状态，避免递归栈溢出风险。

func isSameTree(p *TreeNode, q *TreeNode) bool {
    stack := []struct{ a, b *TreeNode }{{p, q}}

    for len(stack) > 0 {
        top := stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        node1, node2 := top.a, top.b

        if node1 == nil && node2 == nil {
            continue
        }
        if node1 == nil || node2 == nil {
            return false
        }
        if node1.Val != node2.Val {
            return false
        }

        stack = append(stack, struct{ a, b *TreeNode }{node1.Right, node2.Right})
        stack = append(stack, struct{ a, b *TreeNode }{node1.Left, node2.Left})
    }

    return true
}

class Solution {
public:
    bool isSameTree(TreeNode* p, TreeNode* q) {
        stack<pair<TreeNode*, TreeNode*>> stk;
        stk.push({p, q});

        while (!stk.empty()) {
            auto [node1, node2] = stk.top();
            stk.pop();

            if (!node1 && !node2) continue;
            if (!node1 || !node2) return false;
            if (node1->val != node2->val) return false;

            stk.push({node1->right, node2->right});
            stk.push({node1->left, node2->left});
        }

        return true;
    }
};

复杂度分析： - 时间复杂度：O(min(m, n)) - 空间复杂度：O(min(h₁, h₂))，栈深度取决于树高
