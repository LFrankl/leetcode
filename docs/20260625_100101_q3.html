解法一：按位统计法（最优解）

思路：对数字 n 的每一位（个位、十位、百位...），分别统计该位上数字 2 出现的次数。将当前位的高位部分和低位部分分开考虑：当前位小于 2 时，该位出现 2 的次数由高位决定；等于 2 时，还需加上低位+1；大于 2 时，高位需加1再乘以该位的基数。

func count2sInRange(n int) int {
    count := 0
    factor := 1 // 当前位的基数: 1, 10, 100, ...

    for factor <= n {
        lower := n % factor          // 低位部分
        cur := (n / factor) % 10     // 当前位数字
        higher := n / (factor * 10)  // 高位部分

        if cur < 2 {
            count += higher * factor
        } else if cur == 2 {
            count += higher * factor + lower + 1
        } else { // cur > 2
            count += (higher + 1) * factor
        }

        factor *= 10
    }
    return count
}

复杂度：时间 O(log₁₀n)，空间 O(1)

解法二：通用数位DP法

思路：将问题推广为"计算 0~n 中任意数字 d 出现的次数"，使用更通用的公式。本质与解法一相同，但写法更清晰、可扩展，便于处理任意目标数字。

func countDigitInRange(n int, d int) int {
    count := 0
    factor := 1

    for factor <= n {
        lower := n % factor
        cur := (n / factor) % 10
        higher := n / (factor * 10)

        if cur < d {
            count += higher * factor
        } else if cur == d {
            // 当 d==0 时，高位不能全为0，需减去一种全零情况
            if d == 0 {
                if higher == 0 {
                    // 0xx 不存在前导零的情况，跳过
                } else {
                    count += (higher - 1) * factor + lower + 1
                }
            } else {
                count += higher * factor + lower + 1
            }
        } else {
            if d == 0 {
                count += (higher - 1) * factor  // d=0 时高位需额外减1
            } else {
                count += (higher + 1) * factor
            }
        }

        factor *= 10
    }

    // d==0 时，0 本身也算一次
    if d == 0 && n >= 0 {
        count++
    }
    return count
}

// 计算数字2出现的次数
func count2sInRangeDP(n int) int {
    return countDigitInRange(n, 2)
}

复杂度：时间 O(log₁₀n)，空间 O(1)

核心推导举例（n=25，统计十位上的2）： - higher=0, cur=2, lower=5, factor=10 - cur==2 → count += 0×10 + 5+1 = 6（即 20,21,22,23,24,25 十位各出现一次2） - 统计个位上的2：higher=2, cur=5, factor=1 - cur>2 → count += (2+1)×1 = 3（即 2,12,22 个位出现2） - 总计 6+3=9 ✓
