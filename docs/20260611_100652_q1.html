AI 解答 (DeepSeek)

解法一：一次遍历（迭代）

思路：由于链表已排序，重复元素必然相邻。遍历链表时，若当前节点与下一节点值相同，则跳过下一节点（cur.Next = cur.Next.Next），否则正常前进。这样每个值只保留第一次出现的节点。

func deleteDuplicates(head *ListNode) *ListNode {
    cur := head
    for cur != nil && cur.Next != nil {
        if cur.Val == cur.Next.Val {
            cur.Next = cur.Next.Next // 跳过重复节点
        } else {
            cur = cur.Next // 值不同，正常前进
        }
    }
    return head
}

复杂度：时间 O(n)，空间 O(1)

解法二：递归

思路：递归到链表末尾，回溯时判断当前节点与递归返回的头节点值是否相同。若相同，说明当前节点是重复的，应丢弃当前节点，直接返回递归结果；否则保留当前节点。

func deleteDuplicates(head *ListNode) *ListNode {
    if head == nil || head.Next == nil {
        return head
    }
    head.Next = deleteDuplicates(head.Next)
    if head.Val == head.Next.Val {
        return head.Next // 当前节点重复，丢弃
    }
    return head // 当前节点保留
}

复杂度：时间 O(n)，空间 O(n)（递归栈深度）
