3099. 哈沙德数

难度: Easy

标签: 数学

链接: https://leetcode.cn/problems/harshad-number/

题目描述

如果一个整数能够被其各个数位上的数字之和整除，则称之为 哈沙德数（Harshad number）。给你一个整数 x 。如果 x 是 哈沙德数 ，则返回 x 各个数位上的数字之和，否则，返回 -1 。

 

示例 1：

输入： x = 18

输出： 9

解释：

x 各个数位上的数字之和为 9 。18 能被 9 整除。因此 18 是哈沙德数，答案是 9 。

示例 2：

输入： x = 23

输出： -1

解释：

x 各个数位上的数字之和为 5 。23 不能被 5 整除。因此 23 不是哈沙德数，答案是 -1 。

 

提示：

  • 1 <= x <= 100

3️⃣ 代码实现

解法一：直接模拟

func sumOfTheDigitsOfHarshadNumber(x int) int {
    // 计算各位数字之和
    sum := 0
    temp := x
    for temp > 0 {
        sum += temp % 10
        temp /= 10
    }

    // 判断是否为哈沙德数
    if x % sum == 0 {
        return sum
    }
    return -1
}

解法二：使用字符串转换（仅作对比，不推荐）

func sumOfTheDigitsOfHarshadNumber2(x int) int {
    // 将数字转为字符串，遍历每个字符计算和
    s := strconv.Itoa(x)
    sum := 0
    for _, ch := range s {
        sum += int(ch - '0')
    }

    if x % sum == 0 {
        return sum
    }
    return -1
}
