1656. 设计有序流

难度: Easy

标签: 设计, 数组, 哈希表, 数据流

链接: https://leetcode.cn/problems/design-an-ordered-stream/

题目描述

有 n 个 (id, value) 对，其中 id 是 1 到 n 之间的一个整数，value 是一个字符串。不存在 id 相同的两个 (id, value) 对。

设计一个流，以 任意 顺序获取 n 个 (id, value) 对，并在多次调用时 按 id 递增的顺序 返回一些值。

实现 OrderedStream 类：

  • OrderedStream(int n) 构造一个能接收 n 个值的流，并将当前指针 ptr 设为 1 。
  • String[] insert(int id, String value) 向流中存储新的 (id, value) 对。存储后：
    • 如果流存储有 id = ptr 的 (id, value) 对，则找出从 id = ptr 开始的 最长 id 连续递增序列 ，并 按顺序 返回与这些 id 关联的值的列表。然后，将 ptr 更新为最后那个  id + 1 。

    • 否则，返回一个空列表。

 

示例：

输入
["OrderedStream", "insert", "insert", "insert", "insert", "insert"]
[[5], [3, "ccccc"], [1, "aaaaa"], [2, "bbbbb"], [5, "eeeee"], [4, "ddddd"]]
输出
[null, [], ["aaaaa"], ["bbbbb", "ccccc"], [], ["ddddd", "eeeee"]]

解释
OrderedStream os= new OrderedStream(5);
os.insert(3, "ccccc"); // 插入 (3, "ccccc")，返回 []
os.insert(1, "aaaaa"); // 插入 (1, "aaaaa")，返回 ["aaaaa"]
os.insert(2, "bbbbb"); // 插入 (2, "bbbbb")，返回 ["bbbbb", "ccccc"]
os.insert(5, "eeeee"); // 插入 (5, "eeeee")，返回 []
os.insert(4, "ddddd"); // 插入 (4, "ddddd")，返回 ["ddddd", "eeeee"]

 

提示：

  • 1 <= n <= 1000
  • 1 <= id <= n
  • value.length == 5
  • value 仅由小写字母组成
  • 每次调用 insert 都会使用一个唯一的 id
  • 恰好调用 n 次 insert

1️⃣ 题意理解

核心要求： 设计一个有序流系统，系统初始化时指定容量 n。需要实现 insert(id, value) 方法，每次插入一个 (id, value) 对。插入后，如果当前指针 ptr 指向的 id 已经存在值，则返回从 ptr 开始的最长连续 id 序列对应的值列表，并将 ptr 更新为最后一个返回的 id + 1；否则返回空列表。

关键约束： - id 范围是 1 到 n - 每个 id 只会被插入一次 - value 长度固定为 5，仅包含小写字母 - 插入顺序是任意的 - 总共会调用 n 次 insert

边界情况： - n = 1 时的最小规模 - 插入顺序完全逆序（从 n 到 1） - 插入顺序完全正序（从 1 到 n）

输入输出格式： - 构造函数：OrderedStream(n) - 插入方法：insert(id, value) → List

2️⃣ 解题思路

解法一：数组存储 + 指针遍历

核心思想： 使用数组存储所有值，ptr 指针指向下一个应该返回的 id。插入时检查 ptr 位置是否有值，如果有则向后遍历连续有值的 id，收集值并更新 ptr。

解题步骤： 1. 初始化长度为 n+1 的数组（索引 1~n） 2. ptr 初始化为 1 3. insert 时： - 将 value 存入数组对应 id 位置 - 如果 id == ptr，则从 ptr 开始向后遍历，收集连续有值的元素 - 更新 ptr 为最后一个连续 id + 1 - 返回收集的值列表 - 否则返回空列表

注意事项： - 数组索引从 1 开始，需要长度为 n+1 - 连续序列的判断：只要下一个位置有值就继续 - 每次 insert 最多遍历 O(n) 次，但总复杂度是 O(n)

解法二：哈希表存储 + 指针遍历

核心思想： 使用哈希表存储 id-value 映射，其他逻辑与数组解法相同。

解题步骤： 1. 初始化哈希表 map 和 ptr = 1 2. insert 时： - 将 (id, value) 存入 map - 如果 id == ptr，则循环检查 map 中是否存在 ptr - 收集连续存在的值，更新 ptr - 返回结果

注意事项： - 哈希表查找 O(1)，但需要连续检查 - 适合 n 很大但内存有限的情况

3️⃣ 代码实现

解法一：数组存储

type OrderedStream struct {
    data []string  // 存储值的数组，索引从1开始
    ptr  int       // 当前指针
}

func Constructor(n int) OrderedStream {
    return OrderedStream{
        data: make([]string, n+1), // 索引1~n
        ptr:  1,
    }
}

func (this *OrderedStream) Insert(id int, value string) []string {
    // 存储值
    this.data[id] = value

    // 如果当前指针位置没有值，直接返回空
    if this.data[this.ptr] == "" {
        return []string{}
    }

    // 收集从ptr开始的连续有值的序列
    res := []string{}
    for this.ptr < len(this.data) && this.data[this.ptr] != "" {
        res = append(res, this.data[this.ptr])
        this.ptr++
    }

    return res
}

解法二：哈希表存储

type OrderedStream struct {
    data map[int]string
    ptr  int
    n    int
}

func Constructor(n int) OrderedStream {
    return OrderedStream{
        data: make(map[int]string),
        ptr:  1,
        n:    n,
    }
}

func (this *OrderedStream) Insert(id int, value string) []string {
    // 存储值
    this.data[id] = value

    // 如果当前指针位置没有值，直接返回空
    if _, ok := this.data[this.ptr]; !ok {
        return []string{}
    }

    // 收集从ptr开始的连续有值的序列
    res := []string{}
    for this.ptr <= this.n {
        if val, ok := this.data[this.ptr]; ok {
            res = append(res, val)
            this.ptr++
        } else {
            break
        }
    }

    return res
}

4️⃣ 复杂度分析

解法一：数组存储

  • ⏱️ 时间复杂度：
    • 构造函数：O(n) 初始化数组
    • insert 方法：平均 O(1)，最坏 O(n)（当连续序列很长时）
    • 总体：所有 insert 总共遍历数组一次，总时间复杂度 O(n)
  • 💾 空间复杂度：O(n)，用于存储 n 个值

解法二：哈希表存储

  • ⏱️ 时间复杂度：
    • 构造函数：O(1)
    • insert 方法：平均 O(1)，最坏 O(n)（当连续序列很长时）
    • 总体：所有 insert 总共遍历一次所有 id，总时间复杂度 O(n)
  • 💾 空间复杂度：O(n)，哈希表存储 n 个键值对
