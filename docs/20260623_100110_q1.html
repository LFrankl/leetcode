2441. 与对应负数同时存在的最大正整数

难度: Easy

标签: 数组, 哈希表, 双指针, 排序

链接: https://leetcode.cn/problems/largest-positive-integer-that-exists-with-its-negative/

题目描述

给你一个 不包含 任何零的整数数组 nums ，找出自身与对应的负数都在数组中存在的最大正整数 k 。

返回正整数 k ，如果不存在这样的整数，返回 -1 。

 

示例 1：

输入：nums = [-1,2,-3,3]
输出：3
解释：3 是数组中唯一一个满足题目要求的 k 。

示例 2：

输入：nums = [-1,10,6,7,-7,1]
输出：7
解释：数组中存在 1 和 7 对应的负数，7 的值更大。

示例 3：

输入：nums = [-10,8,6,7,-2,-3]
输出：-1
解释：不存在满足题目要求的 k ，返回 -1 。

 

提示：

  • 1 <= nums.length <= 1000
  • -1000 <= nums[i] <= 1000
  • nums[i] != 0

AI 解答 (DeepSeek)

解法一：哈希集合

思路：将所有数字放入哈希集合中，然后遍历数组中的正整数，检查其负数是否也在集合中，记录最大的满足条件的正整数。

func findMaxK(nums []int) int {
    set := make(map[int]bool)
    for _, v := range nums {
        set[v] = true
    }
    maxK := -1
    for _, v := range nums {
        if v > 0 && set[-v] && v > maxK {
            maxK = v
        }
    }
    return maxK
}

复杂度：时间 O(n)，空间 O(n)

解法二：排序 + 双指针

思路：将数组排序后，左指针指向最小值（负数端），右指针指向最大值（正数端）。若两指针所指数字绝对值相等且符号相反，则找到答案；若左指针绝对值更大，左指针右移；否则右指针左移。

func findMaxK(nums []int) int {
    sort.Ints(nums)
    l, r := 0, len(nums)-1
    for l < r {
        if nums[l] < 0 && nums[r] > 0 && -nums[l] == nums[r] {
            return nums[r]
        }
        if -nums[l] > nums[r] {
            l++
        } else {
            r--
        }
    }
    return -1
}

复杂度：时间 O(n log n)，空间 O(1)（不计排序开销）
