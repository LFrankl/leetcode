372. 超级次方

题目描述

你的任务是计算 ab 对 1337 取模，a 是一个正整数，b 是一个非常大的正整数且会以数组形式给出。

 

示例 1：

输入：a = 2, b = [3]
输出：8

示例 2：

输入：a = 2, b = [1,0]
输出：1024

示例 3：

输入：a = 1, b = [4,3,3,8,5,2]
输出：1

示例 4：

输入：a = 2147483647, b = [2,0,0]

提示：

  • 1 <= a <= 231 - 1
  • 1 <= b.length <= 2000
  • 0 <= b[i] <= 9
  • b 不含前导 0

解法一：逐位迭代 + 快速幂

思路：利用指数分解性质：a^(10k+d) = (a^k)^10 × a^d。从左到右遍历 b 的每一位数字，维护当前结果 result = a^(已处理部分)，每遇到新数字 d，更新 result = result^10 × a^d (mod 1337)。配合快速幂计算小指数的模幂。

func superPow(a int, b []int) int {
    const MOD = 1337
    result := 1
    a %= MOD
    for _, digit := range b {
        // result^(10) * a^digit mod MOD
        result = powMod(result, 10, MOD) * powMod(a, digit, MOD) % MOD
    }
    return result
}

// 快速幂：计算 a^b mod m
func powMod(a, b, m int) int {
    result := 1
    a %= m
    for b > 0 {
        if b&1 == 1 {
            result = result * a % m
        }
        a = a * a % m
        b >>= 1
    }
    return result
}

复杂度：时间 O(n·log₁₀)，空间 O(1)，其中 n 为数组 b 的长度，每次迭代做两次快速幂（指数最大为10），每次快速幂 O(log₁₀)=O(1)。

解法二：扩展欧拉定理 + 快速幂

思路：1337 = 7 × 191，φ(1337) = 6 × 190 = 1140。由扩展欧拉定理：当 b ≥ φ(m) 时，a^b ≡ a^(b mod φ(m) + φ(m)) (mod m)，无论 gcd(a,m) 是否为1。先将大数组 b 对 φ(m) 取模得到缩减指数，再判断是否需要加上 φ(m)，最后用一次快速幂求解。

func superPow(a int, b []int) int {
    const MOD = 1337
    const PHI = 1140 // φ(1337) = φ(7) × φ(191) = 6 × 190

    // 计算 b mod PHI
    exp := 0
    for _, digit := range b {
        exp = (exp * 10 + digit) % PHI
    }

    // 判断 b >= PHI，需要加 PHI（扩展欧拉定理）
    if isGreaterOrEqual(b, PHI) {
        exp += PHI
    }

    return powMod(a, exp, MOD)
}

// 判断数组表示的数是否 >= threshold
func isGreaterOrEqual(b []int, threshold int) bool {
    // 将 threshold 转为数字数组: 1140 -> [1,1,4,0]
    digits := make([]int, 0)
    for t := threshold; t > 0; t /= 10 {
        digits = append([]int{t % 10}, digits...)
    }
    if len(b) > len(digits) {
        return true
    }
    if len(b) < len(digits) {
        return false
    }
    for i := 0; i < len(b); i++ {
        if b[i] > digits[i] {
            return true
        }
        if b[i] < digits[i] {
            return false
        }
    }
    return true // 相等
}

// 快速幂
func powMod(a, b, m int) int {
    result := 1
    a %= m
    for b > 0 {
        if b&1 == 1 {
            result = result * a % m
        }
        a = a * a % m
        b >>= 1
    }
    return result
}

复杂度：时间 O(n + log(exp))，空间 O(1)。其中遍历数组取模 O(n)，最终快速幂的指数 exp ≤ φ(m) + φ(m) = 2280，故 O(log(2280)) ≈ O(1)。整体 O(n)。
