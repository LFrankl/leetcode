3083. 字符串及其反转中是否存在同一子字符串

难度: Easy

标签: 哈希表, 字符串

链接: https://leetcode.cn/problems/existence-of-a-substring-in-a-string-and-its-reverse/

题目描述

给你一个字符串 s ，请你判断字符串 s 是否存在一个长度为 2 的子字符串，在 s 反转后的字符串中也出现。

如果存在这样的子字符串，返回 true；如果不存在，返回 false 。

 

示例 1：

输入：s = "leetcode"

输出：true

解释：子字符串 "ee" 的长度为 2，它也出现在 reverse(s) == "edocteel" 中。

示例 2：

输入：s = "abcba"

输出：true

解释：所有长度为 2 的子字符串 "ab"、"bc"、"cb"、"ba" 也都出现在 reverse(s) == "abcba" 中。

示例 3：

输入：s = "abcd"

输出：false

解释：字符串 s 中不存在满足「在其反转后的字符串中也出现」且长度为 2 的子字符串。

 

提示：

  • 1 <= s.length <= 100
  • 字符串 s 仅由小写英文字母组成。

解法一：集合法

思路：关键观察——子字符串 "ab" 在反转字符串中出现，等价于 "ba" 在原字符串中出现（因为反转字符串中的 "ab" 对应原字符串中的 "ba"）。因此问题转化为：原字符串中是否存在某个长度为2的子串 "ab"，使得其反转 "ba" 也出现在原字符串中。特殊地，"aa" 类子串天然满足条件。

func isSubstringPresent(s string) bool {
    seen := make(map[string]bool)
    for i := 0; i < len(s)-1; i++ {
        sub := s[i:i+2]
        seen[sub] = true
        rev := string(sub[1]) + string(sub[0])
        if seen[rev] {
            return true
        }
    }
    return false
}

复杂度：时间 O(n)，空间 O(n)

解法二：位压缩法（最优）

思路：利用26个字母的特点，用一个26×26的布尔矩阵（用int位运算压缩）记录所有出现的2字符组合。对每个子串 "ab"，标记 matrix[a][b]=true，同时检查 matrix[b][a] 是否已为 true。用单个 int 的26位来压缩每行，节省空间。

func isSubstringPresent(s string) bool {
    mask := make([]int, 26)
    for i := 0; i < len(s)-1; i++ {
        a, b := int(s[i]-'a'), int(s[i+1]-'a')
        mask[a] |= 1 << b
        if mask[b]>>a&1 == 1 {
            return true
        }
    }
    return false
}

复杂度：时间 O(n)，空间 O(1)（仅26个int，常数空间）
