难度: Easy

题目描述

给你一个整数数组 hours，表示以 小时 为单位的时间，返回一个整数，表示满足 i < j 且 hours[i] + hours[j] 构成 整天 的下标对 i, j 的数目。

整天 定义为时间持续时间是 24 小时的 整数倍 。

例如，1 天是 24 小时，2 天是 48 小时，3 天是 72 小时，以此类推。

 

示例 1：

输入： hours = [12,12,30,24,24]

输出： 2

解释：

构成整天的下标对分别是 (0, 1) 和 (3, 4)。

示例 2：

输入： hours = [72,48,24,3]

输出： 3

解释：

构成整天的下标对分别是 (0, 1)、(0, 2) 和 (1, 2)。

 

提示：

  • 1 <= hours.length <= 100
  • 1 <= hours[i] <= 109

AI 解答 (DeepSeek)

解法一：暴力枚举

思路：枚举所有满足 i < j 的下标对，判断 hours[i] + hours[j] 是否为 24 的整数倍，即 (hours[i] + hours[j]) % 24 == 0。数据规模 n ≤ 100，暴力完全可行。

func countCompleteDayPairs(hours []int) int {
    count := 0
    for i := 0; i < len(hours); i++ {
        for j := i + 1; j < len(hours); j++ {
            if (hours[i] + hours[j]) % 24 == 0 {
                count++
            }
        }
    }
    return count
}

复杂度：时间 O(n²)，空间 O(1)

解法二：余数计数（最优）

思路：(hours[i] + hours[j]) % 24 == 0 等价于 hours[i] % 24 + hours[j] % 24 == 24（或两者余数均为 0）。先将每个元素对 24 取余，用数组计数余数出现次数，然后对余数 r，找配对余数 (24 - r) % 24 的数量即可。注意余数 0 和 12 的自配对需要特殊处理（组合数 C(cnt, 2)）。

func countCompleteDayPairs(hours []int) int {
    cnt := [24]int{}
    for _, h := range hours {
        cnt[h % 24]++
    }
    // 余数 0 的自配对：C(cnt[0], 2)
    // 余数 12 的自配对：C(cnt[12], 2)
    // 余数 r 与余数 24-r 配对（1<=r<=11）
    res := cnt[0] * (cnt[0] - 1) / 2
    res += cnt[12] * (cnt[12] - 1) / 2
    for r := 1; r <= 11; r++ {
        res += cnt[r] * cnt[24 - r]
    }
    return res
}

复杂度：时间 O(n)，空间 O(1)（余数数组大小固定为 24）
