1️⃣ 题意理解

题目重述
给定一个字符串 word 和一个缩写字符串 abbr，判断 abbr 是否是 word 的有效缩写。
缩写规则： - 可以用数字表示跳过的字符数（数字不能有前导零） - 缩写中的字母必须与 word 中对应位置的字母相同 - 缩写必须完全匹配 word 的所有字符

关键约束 - word 只包含小写字母 - abbr 只包含小写字母和数字 - 数字不能有前导零（但可以是 0 表示跳过 0 个字符） - 缩写必须完全匹配 word 的长度

边界情况 - 数字出现在末尾 - 数字为 0 - 缩写长度可能比 word 长或短 - 连续数字（如 "12" 表示跳过 12 个字符）

输入输出格式 - 输入：string word, string abbr - 输出：bool（是否有效）

2️⃣ 解题思路

解法一：双指针扫描

核心思想
使用两个指针分别遍历 word 和 abbr，遇到数字时解析数字并跳过 word 中对应数量的字符，遇到字母时直接比较。

步骤 1. 初始化指针 i（指向 word）和 j（指向 abbr） 2. 遍历 abbr： - 如果 abbr[j] 是数字： - 解析连续数字（注意前导零） - 移动 i 指针跳过对应数量的字符 - 否则： - 比较 word[i] 和 abbr[j] - 不匹配则返回 false 3. 最后检查 i 是否恰好到达 word 末尾

注意事项 - 数字可能有多位 - 数字为 0 时跳过 0 个字符 - 数字不能有前导零（但 "0" 是允许的）

解法二：正则表达式（理论解法）

核心思想
将 abbr 转换为正则表达式，然后匹配 word。
例如："a2e" → "a.{2}e"
但需要注意数字不能有前导零的约束，且需要完全匹配长度。

步骤 1. 遍历 abbr，构建正则表达式 2. 遇到数字时： - 检查前导零 - 转换为 .{n} 模式 3. 使用正则匹配，检查完全匹配

注意事项 - 需要处理边界条件 - 正则表达式可能效率较低 - 需要额外处理长度匹配

3️⃣ 代码实现

解法一：双指针扫描

func validWordAbbreviation(word string, abbr string) bool {
    i, j := 0, 0 // i 指向 word，j 指向 abbr
    n, m := len(word), len(abbr)

    for i < n && j < m {
        // 如果 abbr[j] 是数字
        if abbr[j] >= '0' && abbr[j] <= '9' {
            // 前导零检查：数字为 0 且后面还有数字时无效
            if abbr[j] == '0' {
                return false
            }

            // 解析数字
            num := 0
            for j < m && abbr[j] >= '0' && abbr[j] <= '9' {
                num = num*10 + int(abbr[j]-'0')
                j++
            }

            // 跳过 word 中的字符
            i += num
            // 如果跳过太多字符，直接返回 false
            if i > n {
                return false
            }
        } else {
            // 如果是字母，直接比较
            if word[i] != abbr[j] {
                return false
            }
            i++
            j++
        }
    }

    // 最后必须同时到达末尾
    return i == n && j == m
}

4️⃣ 复杂度分析

解法一：双指针扫描

  • ⏱️ 时间复杂度：O(n + m)
    其中 n 是 word 长度，m 是 abbr 长度。每个字符最多访问一次。
  • 💾 空间复杂度：O(1)
    只使用了常数级别的额外空间。

常规用例

word = "hello"
abbr = "h01lo"
输出: false
解释: 数字有前导零

// 用例 5：完全匹配
word = "hi"
abbr = "2"
输出: true
解释: 跳过 2 个字符，正好匹配 "hi"
