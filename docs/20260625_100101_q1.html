题目描述

给你一个整数数组 nums ，返回出现最频繁的偶数元素。

如果存在多个满足条件的元素，只需要返回 最小 的一个。如果不存在这样的元素，返回 -1 。

 

示例 1：

输入：nums = [0,1,2,2,4,4,1]
输出：2
解释：
数组中的偶数元素为 0、2 和 4 ，在这些元素中，2 和 4 出现次数最多。
返回最小的那个，即返回 2 。

示例 2：

输入：nums = [4,4,4,9,2,4]
输出：4
解释：4 是出现最频繁的偶数元素。

示例 3：

输入：nums = [29,47,21,41,13,37,25,7]
输出：-1
解释：不存在偶数元素。

 

提示：

  • 1 <= nums.length <= 2000
  • 0 <= nums[i] <= 105

解法一：哈希表计数

思路：遍历数组，用哈希表记录所有偶数元素的出现次数，然后在哈希表中找到出现次数最多的元素，若有多个则取最小的值。若无偶数则返回 -1。

func mostFrequentEven(nums []int) int {
    freq := map[int]int{}
    for _, num := range nums {
        if num % 2 == 0 {
            freq[num]++
        }
    }
    if len(freq) == 0 {
        return -1
    }
    ans, maxFreq := -1, 0
    for num, cnt := range freq {
        if cnt > maxFreq || (cnt == maxFreq && num < ans) {
            ans = num
            maxFreq = cnt
        }
    }
    return ans
}

复杂度：时间 O(n)，空间 O(n)

解法二：排序后遍历计数

思路：先筛选出所有偶数并排序，然后一趟遍历统计连续相同值的个数，同时维护最大频次和最小值答案。排序后相同值自然相邻，遍历更简洁。

func mostFrequentEven(nums []int) int {
    evens := []int{}
    for _, num := range nums {
        if num % 2 == 0 {
            evens = append(evens, num)
        }
    }
    if len(evens) == 0 {
        return -1
    }
    sort.Ints(evens)
    ans, maxFreq := evens[0], 1
    curNum, curFreq := evens[0], 1
    for i := 1; i < len(evens); i++ {
        if evens[i] == curNum {
            curFreq++
        } else {
            if curFreq > maxFreq {
                ans, maxFreq = curNum, curFreq
            }
            curNum, curFreq = evens[i], 1
        }
    }
    // 别忘了最后一组
    if curFreq > maxFreq {
        ans, maxFreq = curNum, curFreq
    }
    return ans
}

复杂度：时间 O(n log n)，空间 O(n)

推荐解法一，哈希表方案时间复杂度更优（O(n)），且代码更简洁。
