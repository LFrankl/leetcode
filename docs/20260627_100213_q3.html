924. 尽量减少恶意软件的传播

难度: Hard

标签: 深度优先搜索, 广度优先搜索, 并查集, 图, 数组, 哈希表

链接: https://leetcode.cn/problems/minimize-malware-spread/

题目描述

给出了一个由 n 个节点组成的网络，用 n × n 个邻接矩阵图 graph 表示。在节点网络中，当 graph[i][j] = 1 时，表示节点 i 能够直接连接到另一个节点 j。 

一些节点 initial 最初被恶意软件感染。只要两个节点直接连接，且其中至少一个节点受到恶意软件的感染，那么两个节点都将被恶意软件感染。这种恶意软件的传播将继续，直到没有更多的节点可以被这种方式感染。

假设 M(initial) 是在恶意软件停止传播之后，整个网络中感染恶意软件的最终节点数。

如果从 initial 中移除某一节点能够最小化 M(initial)， 返回该节点。如果有多个节点满足条件，就返回索引最小的节点。

请注意，如果某个节点已从受感染节点的列表 initial 中删除，它以后仍有可能因恶意软件传播而受到感染。

 

示例 1：

输入：graph = [[1,1,0],[1,1,0],[0,0,1]], initial = [0,1]
输出：0

示例 2：

输入：graph = [[1,0,0],[0,1,0],[0,0,1]], initial = [0,2]
输出：0

示例 3：

输入：graph = [[1,1,1],[1,1,1],[1,1,1]], initial = [1,2]
输出：1

 

提示：

  • n == graph.length
  • n == graph[i].length
  • 2 <= n <= 300
  • graph[i][j] == 0 或 1.
  • graph[i][j] == graph[j][i]
  • graph[i][i] == 1
  • 1 <= initial.length <= n
  • 0 <= initial[i] <= n - 1
  • initial 中所有整数均不重复

解法一：并查集（Union-Find）

思路：先用并查集将所有连通的节点合并。对于每个连通分量，如果其中恰好只有一个初始感染节点，则移除该节点可以"拯救"整个连通分量。我们选择能拯救最多节点的那个初始感染节点；若多个节点拯救数相同，返回索引最小的。

func minMalwareSpread(graph [][]int, initial []int) int {
    n := len(graph)
    parent := make([]int, n)
    size := make([]int, n)
    for i := range parent {
        parent[i] = i
        size[i] = 1
    }

    var find func(x int) int
    find = func(x int) int {
        if parent[x] != x {
            parent[x] = find(parent[x])
        }
        return parent[x]
    }
    union := func(a, b int) {
        pa, pb := find(a), find(b)
        if pa != pb {
            if size[pa] < size[pb] {
                pa, pb = pb, pa
            }
            parent[pb] = pa
            size[pa] += size[pb]
        }
    }

    for i := 0; i < n; i++ {
        for j := i + 1; j < n; j++ {
            if graph[i][j] == 1 {
                union(i, j)
            }
        }
    }

    // 统计每个连通分量中初始感染节点的数量
    infected := make(map[int]int)
    for _, node := range initial {
        root := find(node)
        infected[root]++
    }

    // 对初始列表排序，保证索引小的优先
    sort.Ints(initial)

    ans := initial[0]
    maxSave := 0
    for _, node := range initial {
        root := find(node)
        if infected[root] == 1 {
            if size[root] > maxSave {
                maxSave = size[root]
                ans = node
            }
        }
    }
    return ans
}

复杂度：时间 O(n² · α(n))，空间 O(n)

解法二：BFS 染色

思路：用 BFS/DFS 对每个未访问节点进行扩散，找出所有连通分量及其大小。其余逻辑与并查集一致——只有当某连通分量中恰好含一个初始感染节点时，移除该节点才能拯救该分量全部节点。

func minMalwareSpread(graph [][]int, initial []int) int {
    n := len(graph)
    visited := make([]bool, n)
    color := make([]int, n)
    for i := range color {
        color[i] = -1
    }

    // BFS 染色，求每个连通分量的编号与大小
    colorID := 0
    colorSize := map[int]int{}
    for i := 0; i < n; i++ {
        if !visited[i] {
            queue := []int{i}
            visited[i] = true
            cnt := 0
            for len(queue) > 0 {
                cur := queue[0]
                queue = queue[1:]
                color[cur] = colorID
                cnt++
                for j := 0; j < n; j++ {
                    if graph[cur][j] == 1 && !visited[j] {
                        visited[j] = true
                        queue = append(queue, j)
                    }
                }
            }
            colorSize[colorID] = cnt
            colorID++
        }
    }

    // 统计每个连通分量中初始感染节点数
    infectedCount := map[int]int{}
    for _, node := range initial {
        infectedCount[color[node]]++
    }

    sort.Ints(initial)
    ans := initial[0]
    maxSave := 0
    for _, node := range initial {
        c := color[node]
        if infectedCount[c] == 1 {
            if colorSize[c] > maxSave {
                maxSave = colorSize[c]
                ans = node
            }
        }
    }
    return ans
}

复杂度：时间 O(n²)，空间 O(n)
