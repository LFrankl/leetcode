题目描述

给定一个整数数组 nums 和一个整数 k ，返回其中元素之和可被 k 整除的非空 子数组 的数目。

子数组 是数组中 连续 的部分。

 

示例 1：

输入：nums = [4,5,0,-2,-3,1], k = 5
输出：7
解释：
有 7 个子数组满足其元素之和可被 k = 5 整除：
[4, 5, 0, -2, -3, 1], [5], [5, 0], [5, 0, -2, -3], [0], [0, -2, -3], [-2, -3]

示例 2:

输入: nums = [5], k = 9
输出: 0

 

提示:

  • 1 <= nums.length <= 3 * 104
  • -104 <= nums[i] <= 104
  • 2 <= k <= 104

AI 解答 (DeepSeek)

解法一：前缀和 + 同余定理

思路：利用同余定理——若两个前缀和对 k 取余结果相同，则它们之间的子数组和必能被 k 整除。遍历数组计算前缀和取余，用哈希表统计每个余数出现的次数，每遇到一个余数，答案累加该余数之前出现的次数。注意负数取余需要修正为非负数。

func subarraysDivByK(nums []int, k int) int {
    count := make(map[int]int)
    count[0] = 1 // 前缀和本身就能被 k 整除的情况
    prefixMod, ans := 0, 0
    for _, num := range nums {
        prefixMod = (prefixMod + num%k + k) % k // 修正负数取余
        ans += count[prefixMod]
        count[prefixMod]++
    }
    return ans
}

复杂度：时间 O(n)，空间 O(k)

解法二：前缀和 + 数组代替哈希表（常量优化版）

思路：与解法一完全相同，但由于余数范围固定为 [0, k-1]，可以用长度为 k 的数组替代哈希表，减少哈希开销，实测运行更快。

func subarraysDivByK(nums []int, k int) int {
    modCount := make([]int, k)
    modCount[0] = 1
    prefixMod, ans := 0, 0
    for _, num := range nums {
        prefixMod = (prefixMod + num%k + k) % k
        ans += modCount[prefixMod]
        modCount[prefixMod]++
    }
    return ans
}

复杂度：时间 O(n)，空间 O(k)
