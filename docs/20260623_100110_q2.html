34. 在排序数组中查找元素的第一个和最后一个位置

难度: Medium

标签: 数组, 二分查找

链接: https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/

题目描述

给你一个按照非递减顺序排列的整数数组 nums，和一个目标值 target。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 target，返回 [-1, -1]。

你必须设计并实现时间复杂度为 O(log n) 的算法解决此问题。

 

示例 1：

输入：nums = [5,7,7,8,8,10], target = 8
输出：[3,4]

示例 2：

输入：nums = [5,7,7,8,8,10], target = 6
输出：[-1,-1]

示例 3：

输入：nums = [], target = 0
输出：[-1,-1]

 

提示：

  • 0 <= nums.length <= 105
  • -109 <= nums[i] <= 109
  • nums 是一个非递减数组
  • -109 <= target <= 109

解法一：两次二分查找

思路：由于数组有序且要求 O(log n)，核心是用二分查找分别定位 target 的左边界和右边界。找左边界时，当 nums[mid] >= target 时收缩右边界；找右边界时，当 nums[mid] <= target 时收缩左边界。最后验证找到的位置是否确实等于 target。

func searchRange(nums []int, target int) []int {
    left := findLeft(nums, target)
    if left == -1 {
        return []int{-1, -1}
    }
    right := findRight(nums, target)
    return []int{left, right}
}

func findLeft(nums []int, target int) int {
    lo, hi := 0, len(nums)-1
    idx := -1
    for lo <= hi {
        mid := lo + (hi-lo)/2
        if nums[mid] >= target {
            hi = mid - 1
        } else {
            lo = mid + 1
        }
        if nums[mid] == target {
            idx = mid
        }
    }
    return idx
}

func findRight(nums []int, target int) int {
    lo, hi := 0, len(nums)-1
    idx := -1
    for lo <= hi {
        mid := lo + (hi-lo)/2
        if nums[mid] <= target {
            lo = mid + 1
        } else {
            hi = mid - 1
        }
        if nums[mid] == target {
            idx = mid
        }
    }
    return idx
}

复杂度：时间 O(log n)，空间 O(1)

解法二：二分查找目标值与目标值+1的左边界

思路：更简洁的写法——写一个辅助函数 lowerBound(nums, target) 返回第一个 >= target 的下标。左边界即 lowerBound(nums, target)，右边界即 lowerBound(nums, target+1) - 1。通过验证左边界是否合法来判断 target 是否存在。

func searchRange(nums []int, target int) []int {
    l := lowerBound(nums, target)
    if l == len(nums) || nums[l] != target {
        return []int{-1, -1}
    }
    r := lowerBound(nums, target+1) - 1
    return []int{l, r}
}

func lowerBound(nums []int, target int) int {
    lo, hi := 0, len(nums)
    for lo < hi {
        mid := lo + (hi-lo)/2
        if nums[mid] < target {
            lo = mid + 1
        } else {
            hi = mid
        }
    }
    return lo
}

复杂度：时间 O(log n)，空间 O(1)
