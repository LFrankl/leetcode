# UI 更新说明 - Google 风格优雅界面

## 🎨 设计理念

全新的界面采用 Google Material Design 风格，追求**简洁、优雅、实用**的设计哲学。

## ✨ 主要特性

### 1. **Google 风格色彩**
- **主色调**：纯白背景 + Google 蓝 (#1a73e8)
- **辅助色**：浅灰 (#f8f9fa) + 中性灰边框
- **文字**：深灰主文字 + 中灰次要文字
- **阴影**：Material Design 标准阴影效果

### 2. **默认显示今日内容**
- 打开网站**直接看到今天的题目**
- 不再是列表页，而是完整的题目内容
- 优雅的卡片式布局，每道题一个卡片

### 3. **侧边栏历史记录**
- 点击左上角菜单按钮展开侧边栏
- 显示学习统计（累计题目、连续天数）
- 列出所有历史执行记录
- 点击任意历史记录，主区域切换到对应内容
- 支持 ESC 键快速关闭

### 4. **响应式设计**
- 桌面端：宽松舒适的布局
- 移动端：自适应缩放，触摸友好
- 打印友好：自动隐藏导航和侧边栏

## 🎯 核心功能

### 题目卡片
每道题目显示为一个独立卡片，包含：
- **题目编号和标题**（左侧）
- **难度徽章**（中间）：简单(绿)、中等(黄)、困难(红)
- **LeetCode 链接**（右侧）：直达题目页面
- **完整题目内容**：描述、代码模板、AI 解答

### 侧边栏交互
- **平滑动画**：侧边栏滑入滑出使用 Material Design 曲线
- **遮罩效果**：展开时主区域半透明遮罩
- **自动关闭**：移动端点击历史记录后自动关闭侧边栏
- **键盘支持**：ESC 键快速关闭

### 内容加载
- **动态加载**：通过 JavaScript 异步加载 HTML 文件
- **加载状态**：显示旋转的加载动画
- **错误处理**：优雅地处理加载失败

## 🛠️ 技术实现

### 文件结构
```
docs/
├── index.html           # 主页面（新结构）
├── history.json         # 历史记录数据 (NEW)
├── css/
│   └── style.css        # 全新 Google 风格样式 (NEW)
├── js/
│   └── app.js           # 应用逻辑脚本 (NEW)
└── *.html               # 各个日期的题目 HTML
```

### Python 脚本改动
- ✅ 生成 `history.json` 记录所有历史
- ✅ 每次执行时更新 history.json
- ✅ 保持生成带时间戳的 HTML 文件

### 前端技术
- **纯原生 JS**：无需任何框架，轻量快速
- **CSS 变量**：方便主题定制
- **Flexbox 布局**：灵活响应式
- **Fetch API**：现代化异步加载

## 🎨 颜色方案

```css
--primary-blue: #1a73e8       /* Google 蓝 */
--background: #ffffff          /* 纯白背景 */
--surface: #f8f9fa            /* 浅灰表面 */
--border: #dadce0             /* 边框灰 */
--text-primary: #202124       /* 主文字 */
--text-secondary: #5f6368     /* 次要文字 */
```

### 难度颜色
- **简单**：绿色背景 (#e6f4ea) + 绿色文字 (#137333)
- **中等**：黄色背景 (#fef7e0) + 橙色文字 (#b06000)
- **困难**：红色背景 (#fce8e6) + 红色文字 (#c5221f)

## 📱 响应式断点

- **桌面**：> 768px - 完整布局
- **移动**：≤ 768px - 紧凑布局，侧边栏全屏展开

## 🚀 使用体验

1. **打开网站** → 立即看到今日最新题目
2. **点击左上角菜单** → 查看历史记录
3. **点击任意历史** → 切换到对应日期的题目
4. **点击题目中的链接** → 跳转到 LeetCode 做题
